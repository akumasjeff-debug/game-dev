---

### 初期金流節奏

**第 1 天（教學關）**
- 強制安排 10 組客人，平均客單價 220 元
- 預估總收入：2,200 元
- 食材成本 ~836 元 + 固定支出 1,650 元
- **第1天淨利：約 -286 元（微虧）**
- 教學完後發「開業補貼金 3,000 元」補回

**第 2-7 天：** 客流每天 +1-2 組，第 7 天預估日淨利 500-800 元

**第一次擴建時機：** 約第 10-14 天可達 8,000 元購桌費門檻

---

### 資金不足懲罰機制

| 情境 | 懲罰 |
|------|------|
| 食材採購金不足 | 部分品項「今日售完」，損失潛在收入 |
| 薪資發不出來 | 員工滿意度 -15，效率降低 20% |
| 連續 3 日薪資拖欠 | 員工自動離職，需 2 天重新招聘 |
| 設備超 30 日未維護 | 故障概率每日 +3%，故障當日停業半天 |

不採用負債系統，避免新手壓力過大。

## 二、名聲系統

| 等級 | 名聲區間 | 等級稱號 | 每日基礎客流加成 |
|------|---------|---------|----------------|
| Lv1 | 0–99 | 路邊攤仔 | 0% |
| Lv2 | 100–249 | 巷口人氣店 | +25% |
| Lv3 | 250–499 | 在地熱炒名店 | +60% |
| Lv4 | 500–749 | 夜市炒霸 | +110% |
| Lv5 | 750–899 | 全台灣都知道 | +180% |
| Lv6 | 900–1000 | 熱炒之王 | +280%（+ 觸發媒體事件） |

**爬升速度：**
- Lv1→Lv2：約 7-10 天

- Lv4→Lv5：約 3-4 個月

### 名聲增減機制

**加分**

| 行為 | 名聲加值 |
|------|---------|
| 客人滿意離開（4 星以上） | +2 |
| 客人給 5 星好評 | +5 |
| 完成特別事件 | +15-30 |
| 節日推出限定特餐且客滿 | +20 |
| 員工「服務連擊」（連續 5 組滿意） | +8 |
| 媒體採訪事件（Lv4 以上） | +50-80 |
| 達成菜單里程碑（解鎖第 10 道菜） | +25 |

**扣分**

| 行為 | 名聲扣值 |
|------|---------|
| 客人等待超過閾值 | -3（每超過 1 分鐘再 -1） |
| 送出冷掉的菜 | -5 |
| 食材用完無法出菜 | -8 |
| 差評事件觸發 | -15-25 |
| 強制打烊（資金不足） | -20 |
| 連續 3 天無 5 星評價 | -5/天 |

名聲每日最多扣 30 分（下限保護）。

## 三、組合加成系統（Combo System）

**Combo 1：老灶台套餐**
炭火大灶 + 三杯雞 + 師傅阿義（火候控制 Lv2）
= 三杯雞售價 +12%、「香氣滿滿」特效機率 +20%、名聲加成 ×1.15

**Combo 2：喝到爽快組合**
大冰桶 + 台啤 + 跑堂小美（眼力好，主動補酒）
= 補酒成功率 +40%、酒水收入整體 +18%

**Combo 3：巷口夜市招牌**
手寫粉筆黑板 + 炒米粉（招牌菜）+ 任意員工 3 人以上
= 路過客自動進店率 +15%、炒米粉點餐率 +20%（新客偏好）

**Combo 4：海鮮大王桌**
大型水族箱 + 清蒸魚或炒蛤蜊 + 採購員阿財（市場議價 -12%）
= 海鮮食材成本額外 -8%（共 -20%）、海鮮名聲加成 ×1.3

**Combo 5：老闆娘的眼神**
收銀台升級 + 皮蛋豆腐 + 炒空心菜（套餐組合）+ 老闆娘（精打細算）
= 每日結算誤差率 0%、套餐折扣吸引力 +10%、客人滿意度基準 +3

**Combo 6：深夜人情味**
暖光燈泡 + 薑絲炒大腸 + 熟客阿龍（認人臉，熟客打折 5%）
= 深夜時段（21:00 後）客人停留 +30%、熟客回訪率 +25%、深夜名聲加分 ×1.2

## 四、待遊玩測試後調整的數值

1. 第 1 天淨利是否為負（教學挫折容忍度）
2. 名聲等待時間閾值（依遊戲時間流速重算）
3. Combo 加成疊加上限（建議單項最高 ×2.0）
4. Lv5→Lv6 名聲任務鏈數量與難度
5. 深夜時段實際客流量基數
6. 差評事件觸發頻率（-15 到 -25 在 Lv2 時影響較大）
