## HUD 佈局

### 整體配置

基礎解析度 480x270，HUD 採「上下夾擠、中間留給遊戲場景」策略。

```
┌─────────────────────────────────────────┐
│ [頂部 HUD 28px]                          │
├─────────────────────────────────────────┤
│                                         │
│           遊戲場景區域                   │
│           (480 x 214px)                 │
│                                         │
├─────────────────────────────────────────┤
│ [底部 HUD 28px]                          │
└─────────────────────────────────────────┘
```

### 頂部 HUD（28px 高）

背景：#1A1A2E 半透明（85%），下邊緣加 1px #FF2D55 霓虹線描邊。

```
┌──────────────────────────────────────────────────────────┐
│ [$] 88,880  │ ★★★☆☆  │  第 3 年  8 月  │  晚上 9:30  │ [≡] │
│   金錢(90px)  名聲(70px)  日期(90px)  時間(80px)  選單(32px) │
└──────────────────────────────────────────────────────────┘
```

- **金錢區：** 錢袋圖示（8x8px）+ 數字，色彩 #F5A623，千位加逗號
- **名聲星星區：** 5 顆星，亮星 #F5A623，暗星 #555566
- **日期區：** 「第 N 年 N 月」，白字 #EEEEEE，9px
- **時間區：** 台語時段詞（早上/中午/下午/晚上/凌晨），晚上字色 #AAB4FF
- **漢堡選單鈕：** 三橫線，按下開啟暫停選單

### 底部 HUD（28px 高）

```
┌──────────────────────────────────────────────────────────┐
│ [建造] [擺桌] [雇員] [菜單]  │  提示：客人正在等位置！       │
│   工具列快捷鈕（各 40px）      即時訊息捲動區               │
└──────────────────────────────────────────────────────────┘
```

工具快捷鈕：選中底色 #FF2D55，未選中 #2A2A4A，圖示 16x16px。

## 主選單與暫停選單

### 視覺風格

**台式霓虹燈箱主題**。容器：深色木板底（#2A1A0A），外圍 2px 霓虹管線（#FF2D55），管線外側 2px 暈光（同色 40%）。

標題「台灣熱炒王」使用像素霓虹燈效果，24px，兩側「—★ ★—」裝飾。

### 主選單佈局（全螢幕）

```
┌─────────────────────────────────────┐
│                                     │
│     ══ 台灣熱炒王 ══               │
│       [台式霓虹管字效果]            │
│                                     │
│   ┌─────────────────────────────┐   │
│   │  >  新遊戲                  │   │
│   │     繼續                    │   │
│   │     設定                    │   │
│   │     排行榜                  │   │
│   └─────────────────────────────┘   │
│         版本號 ver 0.1.0            │
└─────────────────────────────────────┘
```

### 按鈕規格

| 狀態 | 底色 | 邊框 | 文字色 | 前綴 |
|------|------|------|--------|------|
| 未選中 | #1A1A2E | 1px #444466 | #CCCCCC | 無 |
| 選中/懸停 | #FF2D55（30%） | 2px #FF2D55 | #FFFFFF | `>` 閃爍游標 |
| 按下瞬間 | #FF2D55（70%） | 2px #FFFFFF | #FFFFFF | `>` |
| 不可用 | #1A1A2E | 1px #333344 | #555566 | 無 |

- 尺寸：200x24px
- `>` 每 500ms 切換顯示/隱藏

### 暫停選單

非全螢幕，半透明遮罩（#000000 60%），選單框 200x180px 居中。
選項：繼續遊戲、存檔、設定、返回主選單。
「返回主選單」需二次確認，文字色 #FF6B6B（警示）。

## 建造模式 UI

### 工具列佈局（底部 HUD 擴展至 48px）

```
┌──────────────────────────────────────────────────────────┐
│ [地板] [牆壁] [桌椅] [廚具] [裝飾] [員工] │ [╳ 離開建造] │
│  32x32  32x32  32x32  32x32  32x32  32x32  │   80px 按鈕  │
└──────────────────────────────────────────────────────────┘
```

各分類按下後向上彈出「子圖示選擇器」（高度 40px，水平捲動）。

### 選中/放置/刪除視覺反饋

| 狀態 | 視覺效果 |
|------|----------|
| 懸停格子 | #FFFFFF 25% 白光覆蓋 |
| 選中設備（可放置） | 底部 #00D26A 2px 外框，有效格子覆蓋淡綠色 |
| 衝突/無法放置 | 設備 50% 紅色 tint，格子 #FF2D55 30%，游標旁顯示 [!] |
| 放置成功 | 縮放動畫（100%→120%→100%，200ms），格子短暫閃白光 |
| 刪除模式 | 游標改為 X 圖示，格子覆蓋 #FF2D55 40%，刪除後設備向下淡出 |

## 對話框與提示

### 訊息對話框

**台式便條紙風格**：米黃底（#FFF8DC），深褐邊框（#6B4226 2px），右上角折角三角，外加 1px 紅色霓虹描邊（#FF2D55）。

```
┌──────────────────────── ◣
│  [圖示 16x16]  標題文字       │
│ ─────────────────────── │
│  內文訊息（最多三行）          │
│                               │
│         [確認]    [取消]       │
└───────────────────────────────┘
```

- 尺寸：280x140px（可依內容調高）
- 標題：12px 粗體，#6B4226
- 內文：10px，#333333
- 彈出動畫：從畫面下方滑入（150ms，ease-out）

### 升級通知彈出

```
╔═══════════════════════════════════╗
║  ★ 等級提升！★                  ║
║                                   ║
║  熱炒攤 → 路邊小館               ║
║  解鎖：新菜單 × 3、員工上限 +2   ║
║                                   ║
║           [ 太好了！ ]            ║
╚═══════════════════════════════════╝
```

- 外框 #F5A623（台啤金），外側 3px 暈光（#F5A623 50%）
- 星號 ★ 每 300ms 閃爍
- 彈出：中心縮放出現（0%→110%→100%，300ms）

建議改為不中斷式 Banner（頂部滑入），玩家點擊後才展開，避免打斷建造操作。

### 確認/取消按鈕規範

| 按鈕類型 | 底色 | 邊框 | 文字色 | 尺寸 |
|----------|------|------|--------|------|
| 確認/好的 | #00D26A | 1px #009A4E | #FFFFFF | 80x22px |
| 取消/返回 | #2A2A4A | 1px #555566 | #CCCCCC | 80x22px |
| 危險操作 | #FF2D55 | 1px #CC1133 | #FFFFFF | 80x22px |

確認鈕永遠在右側，取消在左側（手機慣例）。

### 字體尺寸規範

| 層級 | 場景 | 尺寸（原始） | 縮放至手機 |
|------|------|------------|-----------|
| H1 | 主選單遊戲名稱 | 24px | ~72px |
| H2 | 選單按鈕文字 | 12px | ~36px |
| H3 | 對話框標題 | 12px | ~36px |
| 內文 | 對話框內文 | 10px | ~30px |
| HUD 數值 | 金錢、日期、時間 | 9px | ~27px |
| 最小說明 | 版本號 | 8px | ~24px |

最小可讀下限：8px（縮放後 ~24px），低於此改圖示。

數值顯示使用等寬（monospace）字型，數值變動時加「滾輪動畫」（150ms）。

## UX 問題清單（高風險，交設計組確認）

1. **頂部 HUD 密度**：480px 以下寬度時，時間欄隱藏「時段文字」只保留數字
2. **子圖示捲動誤觸**：一次最多顯示 6 個，超過 6 個改分頁非捲動
3. **刪除操作誤觸**：刪除鈕需長按 0.5 秒才進入刪除模式
4. **升級通知打斷操作**：改為頂部不中斷式 Banner（詳見上方）
5. **「繼續」按鈕無存檔時**：加「尚無存檔」小字，不能只顯示灰色
