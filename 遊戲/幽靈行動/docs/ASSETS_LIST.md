## 五、AI 生成工具比較

### 19. Meshy.ai（3D 模型 AI 生成）
- **類型**：AI 3D 模型生成工具
- **費用**：免費方案 100 credits/月；Pro $20/月（1,000 credits）
- **連結**：https://www.meshy.ai/pricing
- **免費方案細節**：
  - 每月 100 credits，重置日為每月 1 日 00:00 UTC
  - Text to 3D 或 Image to 3D 完整生成需 20 credits（即每月可生成 5 個完整模型）
  - 每月 10 次下載限制（Meshy 6 模型需付費才能下載）
  - 多邊形上限 10,000（高細節需付費）
  - 免費模型授權：CC BY 4.0（商用需標示作者）
- **匯出格式**：FBX、OBJ、USDZ、GLB、STL、BLEND
- **付費模型授權**：完整商業版權，可銷售
- **適配度**：高 — 可從文字或圖片生成 3D 武器、道具、掩體；GLB/FBX 可直接匯入 Godot

### 20. Tripo3D（3D 模型 AI 生成）
- **類型**：AI 3D 模型生成工具
- **費用**：免費方案 200 credits/月（最多 8 個模型）；Pro $13.93/月（3,000 credits）
- **連結**：https://www.tripo3d.ai/pricing
- **免費方案細節**：
  - 每月 200 credits，上限 8 個模型（比 Meshy 免費額度高）
  - 1 個並行任務，標準佇列優先度
  - 最多儲存 10 個模型
  - 免費模型為公開（Public），授權 CC BY 4.0
  - 付費才有私人模型與完整商業版權
- **適配度**：高 — 免費額度比 Meshy 多；生成速度快（Turbo 模式約 10 秒）；幾何較乾淨，適合遊戲網格

### Meshy vs Tripo 建議選擇

| 需求 | 推薦 |
|------|------|
| 每月免費嘗試模型數量多 | Tripo（8 個 vs 5 個） |
| 全流程（生成→貼圖→Rig→動畫）一站式 | Meshy |
| 生成速度快 | Tripo（Turbo 模式） |
| 3D 列印用途 | Meshy（更乾淨的網格） |
| 預算緊，只用免費方案 | Tripo（更多免費 credit） |

### 21. Midjourney（卡牌藝術 / 2D 概念圖）
- **類型**：AI 圖像生成工具
- **費用**：無免費方案；最低 $10/月（Basic，約 3.3 GPU 小時）
- **連結**：https://docs.midjourney.com/hc/en-us/sections/28005319720845-Plan-Information
- **方案對比**：
  - Basic $10/月：3.3 GPU 小時/月
  - Standard $30/月：15 GPU 小時 + 無限 Relax 模式（最具 CP 值）
  - Pro $60/月：30 GPU 小時 + 私人模式
  - Mega $120/月：60 GPU 小時
- **商用條件**：任何付費方案均可商用；年收入超過 $100 萬的公司需 Pro 以上
- **授權**：付費用戶擁有生成圖像的商業使用權
- **適配度**：中 — 用於製作概念藝術、技能卡牌插圖、角色設計稿；不直接作為遊戲 sprite，需轉換

## 六、開發工具

### 22. Blender 5.1（3D 建模 / 動畫）
- **類型**：3D 建模工具
- **授權**：GPL（完全開源免費）
- **費用**：完全免費
- **連結**：https://www.blender.org/download/
- **最新穩定版**：5.1（2026 年 3 月發布）
- **長期支援版**：4.5 LTS（支援至 2027 年 7 月）/ 4.2 LTS（至 2026 年）
- **適配度**：高 — 用於製作等角角色 3D 模型、烘焙等角視角精靈圖、燈光渲染、動畫製作

### 23. Android Studio Quail 1（2026.1.1）
- **類型**：Android 開發 IDE
- **授權**：免費（Apache 2.0）
- **費用**：完全免費
- **連結**：https://developer.android.com/studio
- **最新穩定版**：Quail 1 | 2026.1.1 Patch 2
- **適配度**：高 — Godot 4 匯出 Android APK 需要 Android Studio 提供的 Android SDK 和 Build Tools

### 24. Aseprite（像素藝術動畫編輯器）
- **類型**：像素藝術工具
- **授權**：商業授權（或自行編譯 MIT/GPL 的舊版原始碼）
- **費用**：Steam 售價約 $19.99 USD；自行編譯免費
- **連結**：https://www.aseprite.org/
- **適配度**：高 — 現役最強像素動畫編輯器，有 tilemap 支援、切片、參考圖層、平板壓感；Godot 有 Aseprite Wizard 插件直接匯入 SpriteFrames

### 25. Pixelorama（Aseprite 免費替代品）
- **類型**：像素藝術工具（開源）
- **授權**：MIT（完全開源免費）
- **費用**：完全免費
- **連結**：https://orama-interactive.itch.io/pixelorama
- **平台**：Windows、Linux、macOS、Web 瀏覽器
- **功能**：圖層、洋蔥皮、等角 tilemap 支援、非破壞性效果（陰影/漸層）
- **適配度**：高 — 積極維護（基於 Godot 開發），有等角磚塊模式，完全免費；若預算有限首選此工具

### 26. LibreSprite（不推薦）
- **類型**：像素藝術工具（舊版 Aseprite 分支）
- **授權**：GPL
- **費用**：免費
- **連結**：https://libresprite.github.io/
- **缺點**：維護不活躍；缺少 tilemap、切片、參考圖層、進階平板支援等功能
- **適配度**：低 — 相比 Pixelorama 沒有優勢，不推薦新專案使用

### 27. Tiled Map Editor
- **類型**：地圖編輯器
- **授權**：GPL（開源免費）
- **費用**：免費（可贊助）
- **連結**：https://www.mapeditor.org/
- **適配度**：中 — 支援等角地圖；Godot 可透過 YATI（Yet Another Tiled Importer）插件匯入 .tmx/.tmj 格式
- **備註**：Godot 4.x 已內建 TileMapLayer，等角場景可直接在 Godot 內編輯，Tiled 適合複雜地圖的外部編輯需求
