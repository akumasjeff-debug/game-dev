# 幽靈行動 — 任務地圖設計規格
版本：1.0｜建立日期：2026-06-21

---

## 設計原則

- 地圖從**下往上**呈現，符合小隊推進方向
- 每個節點間距代表約 30–60 秒的自動推進時間
- 決策點是唯一的玩家介入時機，畫面暫停等待選擇
- 岔路左右兩條路最終匯合於相同的下一節點，維持線性主線
- 隱藏事件不在地圖上顯示，僅在帶對角色時觸發

### 各節點詳細規格

#### 節點 1：入口大廳（出發點）
- 敵人：無
- 事件：教學提示（移動說明 + 決策點說明）
- 視覺：現代大廈入口，旋轉玻璃門，警衛台已被清空

#### 節點 2：前台大廳（第一戰鬥房間）
- 敵人：普通警衛 x2（HP 100，ATK 25，DEF 8）
- 決策點：`event_room_entry`（房間入口）
  - 選項 A：直衝進入（無條件）
  - 選項 B：靜悄悄進入（無條件）→ 敵人反應延遲 1.5 秒
  - 說明：教學關第一個決策，只有兩個選項，讓玩家理解決策機制
- 隱藏事件：無

#### 節點 3：開放辦公區（第二戰鬥房間）
- 敵人：普通警衛 x2 + 武裝保全 x1（HP 150，ATK 35，DEF 12）
- 決策點：`event_room_entry`（房間入口）
  - 選項 A：直衝進入（無條件）
  - 選項 B：靜悄悄進入（無條件）→ 敵人反應延遲 1.5 秒
  - 選項 C：投擲炸彈後進入（需有炸彈）→ 所有敵人先扣 60% HP
- 隱藏事件：無

#### 節點 4：岔路（走廊分叉）
- 決策點：`event_fork_office`（岔路）
  - 左路：前往會議室
    - 情報顯示：「較多敵人，但有文件資料，素材獎勵 +50%」
    - 敵人：普通警衛 x3（HP 100，ATK 25）
    - 獎勵：任務素材 x150%
  - 右路：前往茶水間
    - 情報顯示：「敵人較少，有補給箱」
    - 敵人：普通警衛 x2
    - 獎勵：補給箱（補血 / 炸彈 / 抽卡券 三選一）
- 隱藏事件：無（教學關不設隱藏事件）

#### 節點 5：走廊緩衝區
- 敵人：無
- 事件：補給箱（30% 機率觸發 `event_supply_box`）
  - 選項 A：全體補血（回復 20% 最大 HP）
  - 選項 B：補充炸彈（+1 炸彈）
  - 選項 C：取得抽卡券（+1 藍色票）
- 說明：讓玩家在 Boss 前有一個喘息點

#### 節點 6：Boss 房間（副理辦公室）
- 決策點：`event_boss_room`（Boss 房間前）
  - 選項 A：直衝（無條件）→ 正面進入
  - 選項 B：側翼迂迴（無條件）→ 路線較長但初始陣型更有利，全隊攻擊力 +10%
  - 選項 C：引蛇出洞（無條件）→ 引 Boss 出房，在走廊戰鬥，消除護衛的掩體優勢
- Boss：副理陳志明（HP 400，ATK 50，DEF 20）
- 護衛：精英警衛 x1（HP 200，ATK 40，DEF 15）
- 隱藏事件：
  - **高塔狙擊點**（觸發條件：狙擊手 Lv.9）
    - 事件 ID：`event_hidden_sniper_vantage`
    - 效果：Boss 戰前先消滅護衛 x1，護衛不參戰

#### 出口
- 成功撤離，觸發任務結算
- 獎勵：金幣 + 素材 + 藍色票 x1 + 故事片段解鎖

### 各節點詳細規格

#### 節點 1：入口（外圍鐵絲網）
- 敵人：無
- 事件：任務情報顯示（「情報指出倉庫內有重裝保全，注意精英敵人」）

#### 節點 2：外圍警戒區
- 敵人：哨兵 x2（HP 120，ATK 30，DEF 10）
- 事件：無決策點，自動戰鬥
- 說明：第一個接觸點，無需決策，讓玩家感受自動戰鬥節奏

#### 節點 3：第一房間（倉庫入口區）
- 敵人：普通保全 x3（HP 130，ATK 32，DEF 12）
- 決策點：`event_room_entry`（房間入口）
  - 選項 A：直衝進入
  - 選項 B：靜悄悄進入 → 敵人反應延遲
  - 選項 C：投擲炸彈後進入（需有炸彈）
  - 選項 D：舉盾突入（需盾兵 Lv.3）→ 進門期間受傷減半
  - 選項 E：側翼突破（需突擊手 Lv.6）→ 敵人防禦降低 50%

#### 節點 4：第二房間（分揀作業區）
- 敵人：普通保全 x3 + 精英保全 x1（HP 250，ATK 55，DEF 20，有掩體）
- 決策點：`event_room_entry`（房間入口）
  - 選項 A：直衝進入
  - 選項 B：靜悄悄進入
  - 選項 C：投擲炸彈後進入（需有炸彈）→ 消除精英掩體
  - 選項 D：盾牆壓制（需盾兵 Lv.6）→ 清掉 50% 敵人
- 說明：玩家第一次遭遇精英敵人，掩體概念登場

#### 節點 5：岔路 1（走廊分叉）
- 決策點：`event_fork_warehouse_1`（岔路）
  - 左路（走廊 A）：
    - 情報：「已知路線，有 2 名普通警衛」
    - 敵人：哨兵 x2
    - 獎勵：無特殊
  - 右路（走廊 B）：
    - 情報：「未探索，風險未知」
    - 敵人：哨兵 x2（相同，但資訊被隱藏）
    - 獎勵：有機率觸發補給箱
- 隱藏密門（`event_hidden_door`）：
  - 觸發條件：**隊伍中有盾兵 Lv.9**（學會「盾撞破門」）
  - 觸發時機：選擇右路（走廊 B）時，盾兵發現加固鐵門
  - 決策：「盾兵發現一扇加固密門，是否盾撞破門？（盾兵損失 10% HP）」
  - 效果：進入隱藏房間，獲得稀有素材 x3 + 金色票 x1
  - 不觸發：正常通過走廊 B，玩家不知道密門存在

#### 節點 6：第三房間（儲物間）
- 敵人：普通保全 x3 + 精英保全 x1（有增援機制：戰鬥進行 15 秒後追加 x1 普通）
- 決策點：`event_room_entry`（房間入口）
  - 選項 A：直衝進入
  - 選項 B：靜悄悄進入
  - 選項 C：投擲炸彈後進入（需有炸彈）
  - 選項 D：電子干擾（需偵察手 Lv.6）→ 阻斷增援
- 說明：增援機制讓時間壓力出現，偵察手價值體現

#### 節點 7：岔路 2 + 伏擊跡象
- 步驟 1：先觸發 `event_ambush_signs`（伏擊跡象）
  - 偵察手（Lv.1 被動）會提前 5 秒顯示「前方有伏擊跡象」
  - 選項 A：繼續推進（無加成）
  - 選項 B：繞路（路線延長，但繞過伏擊點，不損失 HP）
  - 選項 C：先投煙霧彈（需有炸彈 or 偵察手大招）→ 解除伏擊優勢
- 步驟 2：接著觸發 `event_fork_warehouse_2`（岔路）
  - 左路（卸貨區）：
    - 情報：「有炸藥桶，爆破手有優勢，但敵人較多」
    - 敵人：普通保全 x4（有炸藥桶 x2）
    - 獎勵：素材 +30%
    - 特殊：爆破手自動引爆炸藥桶對所有敵人造成額外傷害
  - 右路（機械室）：
    - 情報：「敵人較少，有武器補給可能」
    - 敵人：普通保全 x3
    - 獎勵：30% 機率武器補給（全隊攻擊力本關 +5%）
    - 隱藏事件：`event_hidden_weapon_cache`（觸發條件：突擊手 Lv.6）→ 武器補給站，全隊攻擊力永久 +5%（本關）

#### 節點 8：第四房間（倉庫深區）
- 敵人：普通保全 x3 + 精英保全 x1（HP 280，ATK 60，DEF 22）
- 決策點：`event_room_entry`（房間入口）
  - 選項 A：直衝進入
  - 選項 B：靜悄悄進入
  - 選項 C：投擲炸彈後進入（需有炸彈）
  - 選項 D：全面制壓（需突擊手 Lv.9）→ 震懾所有敵人 3 秒
- 說明：此房間前預設玩家已有傷亡壓力，自然製造「用大招還是留著 Boss 用」的決策

#### 節點 9：Boss 房間（主管辦公室）
- 決策點：`event_boss_room`（Boss 房間前）
  - 選項 A：直衝
  - 選項 B：側翼迂迴 → 比 Boss 房更有利陣型，全隊攻擊力 +10%
  - 選項 C：引蛇出洞 → 護衛也會一起出來，但取消護衛的掩體護甲
- Boss：張大頭（HP 600，ATK 65，DEF 28，每 30 秒呼叫增援 x1）
- 護衛：精英保全 x1（HP 300，ATK 50，DEF 25）
- 隱藏事件：
  - **高塔狙擊點**（觸發條件：狙擊手 Lv.9）→ Boss 戰前先消滅護衛
  - **傷患急救站**（觸發條件：醫療兵 Lv.9）→ 全隊復活 + 任務後額外金幣

### 各節點詳細規格

#### 節點 1：入口（地下設施入口）
- 敵人：無
- 事件：任務情報（「偵察回報：基地內設有計時爆炸裝置，另有人質被拘禁。帶爆破手會更安全。」）
- 設計意圖：提示玩家此關需要爆破手，同時告知人質任務

#### 節點 2：安全門緩衝
- 敵人：巡邏兵 x2（HP 160，ATK 40，DEF 15）
- 事件：無決策，自動戰鬥
- 視覺：兩道安全門之間的狹窄走廊

#### 節點 3：外圍走廊（必觸發伏擊）
- 敵人：精銳士兵 x3（HP 200，ATK 45，DEF 18）
- 決策點：`event_ambush_signs`（伏擊跡象，必觸發）
  - 選項 A：繼續推進（全隊在伏擊開始時各扣 15% HP）
  - 選項 B：繞路（需額外通過一個小遭遇戰，但不吃伏擊傷害）
  - 選項 C：先投煙霧彈（需偵察手大招 or 炸彈）→ 解除伏擊，敵人失去先手
- 說明：此關的伏擊是必觸發，不是機率事件，強化關卡記憶點

#### 節點 4：第一房間（警衛休息室）
- 敵人：精銳士兵 x4（HP 180，ATK 42，DEF 16）
- 決策點：`event_room_entry`（房間入口）
  - 選項 A：直衝進入
  - 選項 B：靜悄悄進入（警衛休息中，反應延遲 2 秒，比普通靜悄悄更長）
  - 選項 C：投擲炸彈後進入（需有炸彈）
  - 選項 D：連環爆破（需爆破手 Lv.9）→ 此房與後兩房連動炸彈設定

#### 節點 5：第二房間（安全室）
- 敵人：精銳士兵 x4 + 精英指揮官 x1（HP 350，ATK 70，DEF 28，有護盾裝置）
- 決策點：`event_room_entry`（房間入口）
  - 選項 A：直衝進入
  - 選項 B：靜悄悄進入
  - 選項 C：投擲炸彈後進入（需有炸彈）→ 破壞護盾裝置
  - 選項 D：穿透狙擊（需狙擊手 Lv.9）→ 子彈穿透護盾裝置，精英立即失去護盾
  - 選項 E：盾牆壓制（需盾兵 Lv.6）→ 清掉 50% 敵人

#### 節點 6：第三房間（武器倉庫）
- 敵人：精銳士兵 x4 + 精英衛兵 x1（HP 330，ATK 68，DEF 26）
- 決策點：`event_room_entry`（房間入口）
  - 選項 A：直衝進入
  - 選項 B：靜悄悄進入
  - 選項 C：投擲炸彈後進入（需有炸彈）
  - 選項 D：煙霧突破（需爆破手 Lv.6）→ 全隊進入時敵人命中率降低 70%
- 隱藏事件：`event_hidden_explosives_room`（觸發條件：爆破手 Lv.9）
  - 爆破手發現隱藏爆炸物密室，可取得額外炸彈 x3 + 後三場大招 CD 縮短 40%

#### 節點 7：岔路 1
- 決策點：`event_fork_base_1`（岔路）
  - 左路（武裝走廊）：
    - 情報：「直接通道，全副武裝士兵」
    - 敵人：精銳士兵 x5（HP 180，ATK 42，DEF 16）
    - 獎勵：無特殊，但路線較短
  - 右路（實驗室）：
    - 情報：「未探索，可能有研究補給」
    - 敵人：精銳士兵 x3 + 科學人員 x1（HP 100，ATK 20，DEF 5，不攻擊）
    - 獎勵：實驗室補給（補血 or 特殊道具「強化劑」：全隊 ATK+15% 持續 60 秒）
  - 隱藏事件：`event_hidden_intel_base`（觸發條件：偵察手 Lv.9）
    - 選擇右路後，偵察手發現廢棄情報基地
    - 效果：揭露本任務剩餘所有房間敵人配置 + 本次任務雙倍素材

#### 節點 8：計時炸彈（強制特殊事件）
- 敵人：無（此節點純事件節點）
- 決策點：`event_timed_bomb`（計時炸彈，強制觸發，倒數 30 秒）
  - 選項 A：爆破手拆除（需隊伍中有爆破手）→ 耗時 10 秒，期間全隊無法移動
    - 成功：繼續推進，並額外獲得「情報晶片」（任務後換取素材）
    - 失敗（30 秒後未拆）：爆炸，全員倒下，任務失敗
  - 選項 B：全隊撤離（無條件）→ 放棄任務，跳回任務選擇畫面，不算失敗
  - 說明：若隊伍中無爆破手，玩家只能選撤離或豁出去賭（實際上沒有第三個選項，設計上要讓玩家吃一次教訓，下次帶爆破手）
- 視覺：紅色計時器、爆破裝置、閃爍的螢光燈
- 設計意圖：此事件是本關最強的「帶對人才能過」設計，驅動角色陣容策略性

#### 節點 9：第四房間（控制中心）
- 敵人：精銳士兵 x4 + 精英指揮官 x1（HP 380，ATK 75，DEF 30）
- 決策點：`event_room_entry`（房間入口）
  - 選項 A：直衝進入
  - 選項 B：靜悄悄進入
  - 選項 C：投擲炸彈後進入（需有炸彈）
  - 選項 D：全面制壓（需突擊手 Lv.9）→ 震懾所有敵人 3 秒
  - 選項 E：隱形滲透（需偵察手 Lv.9）→ 全隊匿蹤 8 秒，清理前半段敵人無傷

#### 節點 10：岔路 2
- 決策點：`event_fork_base_2`（岔路）
  - 左路（指揮通道）：
    - 情報：「最短路線，但敵人最多且全為精銳」
    - 敵人：精銳士兵 x5（HP 200，ATK 48，DEF 20）
    - 獎勵：不繞路，直達人質房間
  - 右路（維修通道）：
    - 情報：「有爆炸物密室的線索，路線較長」
    - 敵人：精銳士兵 x4
    - 獎勵：路過時有機率觸發補給箱

#### 節點 11：人質房間（特殊事件）
- 敵人：看守士兵 x2（HP 160，ATK 40，DEF 15）
- 決策點：`event_hostage_rescue`（人質救援，戰鬥結束後觸發）
  - 選項 A：救援人質（無條件）
    - 人質跟隨小隊，移動速度降低 20%（推進較慢）
    - 成功撤離後獲得豐厚額外獎勵（金幣 +200%、素材 +100%）
    - 失敗條件新增：人質死亡 → 任務失敗
  - 選項 B：忽略人質，繼續推進
    - 移動速度正常
    - 無額外獎勵
- 說明：此事件引入了「護送降低容錯」的風險/獎勵抉擇

#### 節點 12：Boss 房間（指揮室）
- 決策點：`event_boss_room`（Boss 房間前）
  - 選項 A：直衝（全隊攻擊正面兩個護衛）
  - 選項 B：側翼迂迴（路線較長但陣型更佳，全隊攻擊力 +10%）
  - 選項 C：引蛇出洞（引兩個護衛先出，逐一擊破後再入）
- Boss：幽靈指揮官（HP 900，ATK 90，DEF 35，每 25 秒呼叫增援 x1 精銳）
- 護衛：精英衛兵 x2（HP 400，ATK 75，DEF 30，各有掩體）
- 隱藏事件：
  - **高塔狙擊點**（觸發條件：狙擊手 Lv.9）→ Boss 戰前先消滅護衛 x1（仍剩 x1）
  - **傷患急救站**（觸發條件：醫療兵 Lv.9）→ 全隊復活 + 任務後額外金幣
  - **廢棄情報基地**（觸發條件：偵察手 Lv.9，若節點 7 未觸發）→ 揭露 Boss 的攻擊模式
