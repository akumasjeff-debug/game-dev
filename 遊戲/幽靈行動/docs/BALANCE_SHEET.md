## 一、升等費用曲線（所有職業共用）

**設計原則：**
- 前期（Lv.1~5）：便宜，讓玩家快速感受解鎖，初期 80 顆素材即可升一級
- 後期（Lv.6~10）：昂貴，Lv.8→9 一口氣要 3220 顆，製造長期目標
- 里程碑 Lv.3/6/9：費用比前一等級高約 1.5 倍，配合「解鎖了重要東西」的感受

**費用公式（基礎）：** `round(50 × 1.6^(lv-1))`，里程碑 ×1.5

| 升等 | 本次費用 | 累計費用 | 備注 |
|------|---------|---------|------|
| Lv.1 → 2 | 80   | 80    | 入門門檻，極低 |
| Lv.2 → 3 | 190  | 270   | **里程碑** — 解鎖決策選項 1 |
| Lv.3 → 4 | 205  | 475   | 平緩過渡 |
| Lv.4 → 5 | 330  | 805   | 開始感受壓力 |
| Lv.5 → 6 | 600  | 1405  | **里程碑** — 解鎖決策選項 2（調降自 785，減少峭壁感） |
| Lv.6 → 7 | 700  | 2105  | 後期正式開始（調降自 840，拉開喘息空間） |
| Lv.7 → 8 | 1340 | 3445  | 高投入區 |
| Lv.8 → 9 | 3220 | 6665  | **里程碑** — 最終決策 + 隱藏事件，付出最高 |
| Lv.9 → 10 | 3440 | 10105 | 最高等，彰顯稀有感 |

> **調整依據：** Lv.5→6 原費用 785，跳幅 2.38 倍，Lv.5→7 累計達 1625（Lv.3→5 的 3.28 倍），玩家在里程碑後無喘息空間。調降至 600/700 後，Lv.5→7 累計 1300，減少約 20% 緊繃感，Lv.7 以後高費區間不受影響。

## 七、計算公式速查

```
升等費用（基礎）= round(50 × 1.6^(lv-1))
里程碑修正      = 基礎費用 × 1.5   [lv = 3, 6, 9]

屬性成長        = base × growth^(lv-1)

大招效果乘數    Lv.1=1.00 / Lv.3=1.20 / Lv.6=1.40 / Lv.9=1.60
CD縮短比例      Lv.1=1.00 / Lv.3=0.85 / Lv.6=0.70 / Lv.9=0.50
中間等級（2/4/5/7/8/10）= 前後里程碑線性插值

突擊手有效倍率  = 1 + atk_bonus × (duration / cd)
CC 無敵窗口比率 = stun_duration / cd
爆破手傷害      = 敵人當前HP × hp_damage_ratio（注意是當前HP，非最大HP）
```
