## 三、Demo 驗收清單

全部 ✅ 才能發布。

### 基地流程

- [ ] 開啟遊戲顯示基地畫面（非黑畫面）
- [ ] 頂部金幣數字正確顯示（首次為 0）
- [ ] 任務板顯示「辦公大樓清查」任務卡片
- [ ] 可以點「選擇任務」並看到按鈕切換為「已選取」
- [ ] 職業選擇區顯示 6 個職業按鈕（盾兵、醫療兵、突擊手、狙擊手、爆破手、偵察手）
- [ ] 點擊職業按鈕可加入陣容（第 4 個滿後不再新增，替換最後一個）
- [ ] 點擊陣容槽位可移除該角色
- [ ] 未選滿 4 人按「出發」→ 顯示錯誤提示「請先選滿 4 名隊員！」
- [ ] 選滿 4 人後按「出發執行任務」→ 進入任務場景

### 任務場景初始化

- [ ] 畫面顯示地圖通道（深色路徑）及標示（起點、終點、房間A、補給箱、房間B/Boss）
- [ ] 小隊 4 個角色顯示在起點下方（有顏色方塊）
- [ ] HUD 底部顯示 4 張角色卡（含職業色圓圈、名稱、HP 條、大招狀態）
- [ ] 頂部進度條顯示「進度 0%」
- [ ] 職業選擇正確傳入：基地選的 4 名角色 = 任務場景出現的 4 個角色

### 任務推進流程

- [ ] 小隊從起點自動向上移動（不需玩家操控）
- [ ] 進度條隨推進更新百分比
- [ ] 抵達房間A 觸發區（y~1250）→ 決策面板彈出、小隊停止移動
- [ ] 決策面板顯示標題「前方發現房間」及描述文字
- [ ] 面板顯示 3 個選項按鈕（直衝進入、靜悄悄進入、投擲炸彈）
- [ ] 點擊任一選項後面板關閉、小隊恢復移動

### 傷害系統

- [ ] 選「直衝進入」→ 全隊 HP 下降（角色卡片血條縮短）
- [ ] HP 降到 50% 以下血條變黃、25% 以下變紅
- [ ] HP 百分比數字即時更新

### 大招系統

- [ ] 大招就緒時卡片下區顯示「大招就緒」（綠字）且邊框脈衝橙色
- [ ] 點擊角色卡片下半部 → 觸發大招
- [ ] 大招觸發後切換為 CD 倒數顯示（灰色遮罩 + 秒數）
- [ ] 醫療兵大招：全隊 HP 明顯回升（+30% 最大 HP）
- [ ] 盾兵大招：接下來 5 秒受到的傷害減半（視覺可感知）
- [ ] 突擊手大招：接下來 8 秒攻擊傷害 +60%（數值面感知）
- [ ] 倒數剩餘 5 秒以下字體放大、顯示橙色
- [ ] CD 歸零後恢復「大招就緒」狀態並播放提示音（ult_ready）

### 岔路與後續決策

- [ ] 抵達岔路觸發點（y~970）→ 決策面板彈出
- [ ] 岔路選項包含「左路（困難）」「右路（簡單）」「探索未知路線」
- [ ] 盾兵 Lv.1（正確數值）時，面板「不顯示」舉盾突入選項
- [ ] 選擇後小隊沿對應路徑繼續前進

### 失敗流程

- [ ] 所有角色 HP 歸零後顯示「任務失敗」結果面板（紅字）
- [ ] 結果描述顯示失敗原因文字
- [ ] 結果面板有「返回基地」按鈕（顯示為「重試」）
- [ ] 點擊按鈕成功返回基地場景

### 勝利流程

- [ ] 小隊到達終點觸發點（y~120）→ 顯示「任務完成！」結果面板（綠字）
- [ ] 結果描述包含「獲得 200 金幣」文字
- [ ] 金幣實際加入（SaveManager.coins +200）
- [ ] 結果面板按鈕文字為「返回基地」
- [ ] 點擊按鈕成功返回基地場景

### 基地回流

- [ ] 返回基地後頂部金幣數字增加（勝利獎勵 200 金幣）
- [ ] 陣容選擇保持上一局選擇
- [ ] 可以重新按「出發執行任務」再次進入任務

## 四、Demo 前清理清單（由程式設計師執行）

### 必做（阻擋正確 Demo 體驗）

1. **`scripts/main.gd` 第 59 行**：將盾兵 `"level": 3` 改回 `"level": 1`
   - 目前盾兵預設 Lv.3，所有房間觸發點都會額外顯示「舉盾突入」選項，與 Demo 新手體驗設計不符

2. **`scripts/save_manager.gd`**：新增 `tutorial_completed` 屬性（預設 `false`）並在 `save_game()`/`load_game()` 中讀寫
   - 目前 TutorialManager 讀寫一個不存在的屬性，教學完成狀態無法正確儲存，每次重啟都會重跑教學

### 建議移除（Demo 前清理，不阻擋但影響專業度）

3. **`scripts/main.gd` 第 74 行**：移除 `print("[Main] 啟動任務: ...")` 或包進 `if OS.is_debug_build():` 條件

4. **`scripts/decision_panel.gd` 第 94 行**：移除 `print("[狙擊手大招] ...")` 或包進 debug 條件
