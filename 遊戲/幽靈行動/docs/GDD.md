# 幽靈行動 — 遊戲設計文件（GDD）
版本：2.1｜最後更新：2026-06-21

---

## 一、遊戲定位

| 項目 | 內容 |
|------|------|
| 遊戲類型 | 戰術指揮 + Idle Meta + 抽卡 |
| 參考對象 | 劍與遠征（Idle/Meta 層）、FTL（事件決策）、Clash of Clans（斜角視覺） |
| 視角 | Isometric 斜角俯視 |
| 平台 | Android / iOS 優先，PC 次之 |
| 遊玩人數 | 單人 |
| 引擎 | Godot 4 |
| 核心樂趣 | 自動推進中的「決策中斷」——玩家不管移動，只管在關鍵時刻做出正確判斷 |

### 一句話定位

> 一支特戰小隊自動推進，玩家只在事件發生時介入決策。升等的角色解鎖更多選項，高等角色更能在特定任務觸發隱藏事件——帶對人、才能開那扇門。

## 二、核心循環

### 外層循環（Idle Meta）

```
基地（離線產金幣）
    ↓ 慢速自動累積
任務選擇（主線 / 支線）
    ↓
戰場執行
    ↓
成功 → 獲得素材 + 抽卡資源
失敗 → 無獎勵，基地繼續產金幣，可立即重試
    ↓
用資源：抽卡（獲得新角色）/ 升等（解鎖新決策）
    ↓
回到任務選擇
```

**關鍵設計原則：**
- 失敗不懲罰：失敗只是「沒拿到這次獎勵」，基地繼續運作
- 任務才是主要素材來源，離線金幣只供基礎維持
- 抽卡資源必須靠任務取得，不可純離線刷出

### 內層循環（戰場）

```
從 6 個職業中選擇 4 名隊員出發
    ↓
小隊從畫面下方自動往上推進（斜角視角）
    ↓
遭遇敵人 → 自動交戰（各角色依職業預設行為執行）
    ↓
遇到決策點 → 畫面暫停，玩家選擇
    ↓
執行選擇 → 繼續推進
    ↓
到達終點（成功）或全員倒下（失敗）
```

**戰場節奏感：**
- 玩家絕大多數時間「觀看」自動推進，節奏輕鬆
- 決策點是唯一的主動介入，干淨俐落
- 決策間隔約 30–60 秒，不讓玩家分神卻也不讓玩家無聊

## 三、任務系統

### 任務類型

| 類型 | 說明 | 主要獎勵 |
|------|------|---------|
| 主線任務 | 推進故事，固定難度，線性解鎖 | 故事進度 + 大量素材 |
| 支線任務 | 特定素材或抽卡券，可重複接 | 升卡材料 / 抽卡券 |

### 任務板邏輯

- 基地任務板同時顯示數個可選任務
- 打不過主線 → 刷支線取得素材強化角色 → 再挑主線
- 任務前：2–3 行文字情報（任務目標 + 場景背景）
- 任務後：成功畫面列出獲得的資源；主線有解鎖故事片段

## 四、角色系統

### 獲得方式

- 所有角色透過**抽卡**取得
- 抽卡資源：藍色票（支線任務）、金色票（主線任務 / 活動）
- 重複獲得同一角色 → 轉化為該角色的「碎片」，累積後可強化稀有度

### 升等效益（三個層面同時成長）

升等花費「角色素材」（任務掉落），每次升等帶來：

| 升等效益 | 說明 |
|---------|------|
| 數值成長 | HP、攻擊、防禦提升 |
| 大招強化 | 大招效果提升 + CD 縮短 |
| 里程碑解鎖 | 特定等級解鎖新決策選項 |

```
Lv.1：加入 → 自動行為 + 大招可用（基礎效果 + 長 CD）
Lv.3：大招效果 +20%，CD 縮短 15%，解鎖決策選項 1
Lv.6：大招效果 +40%，CD 縮短 30%，解鎖決策選項 2
Lv.9：大招效果 +60%，CD 縮短 50%，解鎖決策選項 3 + 隱藏事件技能
```

**設計意圖：** 同樣的關卡，Lv.1 和 Lv.9 的角色打法完全不同；升卡讓玩家感覺「這張卡真的變強了」——不只是數字，是節奏和能力都改變。

### 大招系統（卡片點擊）

任務進行中，玩家可以隨時**點擊畫面底部的角色卡片**施放大招：

- 大招立即生效，不暫停遊戲
- 施放後進入 CD，CD 結束前卡片變暗無法點擊
- 各職業 CD 不同，製造「管理多張卡 CD」的節奏感

| 職業 | 大招 | 大招類型 | 基礎效果 | 基礎 CD |
|------|------|---------|---------|---------|
| 盾兵 | 防禦護盾 | 防禦提升 | 全隊受傷害降低 50%，持續 5 秒 | 30 秒 |
| 醫療兵 | 緊急治療 | 血量恢復 | 全隊立即恢復 30% 最大 HP | 40 秒 |
| 突擊手 | 火力全開 | 攻擊提升 | 全隊攻擊力提升 60%，持續 8 秒 | 25 秒 |
| 狙擊手 | 精準鎖定 | 條件瞬殺（單體）| 指定一個敵人：若其 HP 低於 25% 則瞬殺，否則造成 300% 攻擊力傷害 | 35 秒 |
| 爆破手 | 引爆炸彈 | 集體瞬間傷害（AOE）| 房間內所有敵人立即扣 70% HP | 45 秒 |
| 偵察手 | 煙霧封鎖 | 控制（CC）| 投擲煙霧彈，所有敵人攻擊失效 5 秒 | 35 秒 |

## 六、決策點設計

決策點是遊戲的核心互動，觸發時**畫面暫停**，玩家做選擇後繼續。

### 決策點類型

#### 一般事件（最常見）

小隊推進過程中隨機觸發，包含：進入房間、岔路、遭遇事件。
基本進入選項無需特殊條件，角色技能解鎖的選項另外標示。

| 事件 | 基本選項 |
|------|---------|
| 前方房間 | 直衝進入 / 靜悄悄進入 / 投擲炸彈後進入 |
| 岔路 | 左路（情報顯示難易）/ 右路 / 未探索路線 |
| 伏擊跡象 | 繼續推進 / 繞路 / 先投煙霧彈 |

#### 技能解鎖的隱藏事件

某些任務會出現**只有帶對角色才能觸發的隱藏事件**，完成後給予特殊獎勵。
玩家事先看不到這些事件——只有隊伍中有對應技能的角色，才會在推進途中「發現」它。

| 隱藏事件 | 觸發條件 | 獎勵 |
|---------|---------|------|
| 加固密門 | 盾兵 Lv.9 學會「盾撞破門」 | 隱藏房間：稀有素材 / 高級抽卡券 |
| 武器補給站 | 突擊手 Lv.6 學會「側翼突破」 | 隱藏武器庫：全隊攻擊力永久 +5%（本關） |
| 爆炸物密室 | 爆破手 Lv.9 學會「連環爆破」 | 取得額外炸彈 x3，下三場大招 CD 縮短 40% |
| 傷患急救站 | 醫療兵 Lv.9 學會「強化血清」 | 全隊復活 + 任務後額外金幣 |
| 高塔狙擊點 | 狙擊手 Lv.9 學會「穿透狙擊」 | Boss 戰前先消滅一個 Boss 護衛 |
| 廢棄情報基地 | 偵察手 Lv.9 學會「幽靈撤離路線」 | 發現敵方作戰計畫：揭露本任務剩餘所有房間的敵人配置與決策點類型，同時獲得本次任務雙倍素材獎勵 |

**設計意圖：** 同一任務，不同隊伍組合看到的事件不同。高等角色讓玩家解鎖「隱形的第二層內容」，驅動抽卡與升等動機。

#### 岔路

小隊到達分叉通道時：

| 選項 | 說明 |
|------|------|
| 左路 | 顯示「已知情報」（例：較多敵人 / 有補給箱） |
| 右路 | 顯示「已知情報」（例：較少敵人 / 較少獎勵） |
| 未探索路線 | 無情報，風險與獎勵均未知 |

#### 特殊事件

| 事件類型 | 決策內容 |
|---------|---------|
| 伏擊 | 發現敵人埋伏跡象：繼續推進 / 原路撤退 / 投入煙霧彈壓制 |
| 人質 | 發現被俘人質：救援（護送至出口）/ 忽略繼續推進 |
| 計時炸彈 | 發現計時裝置：爆破手拆除（耗時 10 秒，期間無法移動）/ 全隊撤離 |
| 補給箱 | 發現補給：全體補血 / 補充炸彈 / 取得抽卡券（三選一） |
| Boss 房間前 | 進入前選擇：直衝 / 側翼迂迴（更長路線但更有利陣型）/ 引蛇出洞 |

## 七、戰場自動行為規則

### 隊伍組成

每次任務從 6 個職業中選 4 人出戰。選人本身就是策略——不同組合應對不同任務類型，強化重複遊玩動機：

- 高難度主線：優先帶盾兵 + 醫療兵保命
- 限時任務：帶突擊手 + 爆破手快速清房
- 情報任務：帶偵察手提早揭露決策點
- Boss 前：狙擊手點殺 Boss 護衛拉高容錯

### 各職業自動行為

各職業的自動行為在無決策點時**全自動執行**，玩家無需介入：

| 職業 | 隊列位置 | 攻擊優先目標 | 特殊自動行為 |
|------|---------|------------|------------|
| 盾兵 | 最前（第 1 位） | 最近的敵人 | 自動格擋正面傷害 |
| 突擊手 | 前中（第 2 位） | 最近的威脅 | 連射不停，攻速最高 |
| 爆破手 | 中間（第 3 位） | 中距離敵人 | 自動引爆炸藥桶 |
| 醫療兵 | 中後（第 3 或 4 位）| 不主動攻擊 | 自動治療 HP 最低的隊員 |
| 狙擊手 | 後方（倒數第 2 位）| 最遠的高威脅敵人 | 自動點殺遠距離目標 |
| 偵察手 | 最後（最末位）| 不主動攻擊 | 持續偵察前方，機率觸發決策點預警 |

**注意：** 實際隊列由選出的 4 名成員動態排列，依職業定位自動就位。偵察手若在隊中，永遠佔據最末位，讓狙擊手前移一格。

## 八、勝負條件

### 成功條件

- 小隊到達關卡終點（依任務類型：撤離出口 / 消滅目標 / 確保目標）

### 失敗條件

- 全員倒下（全滅）
- 特定任務的特殊條件（例：人質死亡）

### 失敗後果

- 無金幣懲罰、無裝備損失
- 基地繼續離線產金幣
- 可立即重試（相同陣容或換人）

## 九、基地系統

### 基地功能

| 功能 | 說明 |
|------|------|
| 離線金幣生產 | 每小時自動產出一定量的金幣（慢速，維持感而非主要收入） |
| 任務板 | 選擇主線 / 支線任務 |
| 角色管理 | 查看、升等角色，從 6 個職業中選 4 人分配出戰陣容（4 個槽） |
| 抽卡系統 | 用票券抽取新角色 |
| 道具倉庫 | 管理炸彈、醫療包等戰場消耗品庫存 |

### 基地離線金幣規格

- 基礎產出：每小時 100 金幣
- 無離線上限（讓玩家長期不玩也有小回報）
- 金幣用途：升等費用、消耗品補充

## 十、美術方向

### 視覺風格

| 層次 | 說明 |
|------|------|
| 整體視角 | Isometric 斜角俯視（固定視角，類 Clash of Clans） |
| 戰場角色 | 簡易 3D 模型，Meshy.ai 生成後匯入 Godot 4，保持低多邊形風格 |
| 背景環境 | 簡易 3D 場景，幾何建築為主，顏色對比清晰 |
| 抽卡 / 角色頁面 | Midjourney 精緻插畫，高品質卡牌藝術風格 |
| UI | 軍事風格介面，深色調 + 橄欖綠 + 橙色重點色 |

### 色彩規範

- 主色：深軍綠（#1A2B1A）、深灰（#2C2C2C）
- 重點色：橙色（#E8600A）、白色（#F0F0F0）
- 危險狀態：紅色（#CC2222）
- 成功狀態：亮綠（#44CC44）

### 角色視覺辨識度

每個職業在戰場上需要一眼辨識：
- 盾兵：體型最大、有明顯大型防彈盾牌
- 突擊手：標準護甲、突擊步槍、快速移動姿態
- 爆破手：背包最大、有炸藥視覺元素、護目鏡
- 醫療兵：白色臂章標誌、無重武器
- 狙擊手：最纖細、長槍輪廓、偽裝披風
- 偵察手：最輕裝、夜視鏡頭戴、肩掛偵察鏡、無顯眼武器

## 十二、音效與音樂方向

| 類別 | 方向 |
|------|------|
| BGM | 低沉軍事電子樂，任務中偏緊張，基地偏沉穩 |
| 決策點音效 | 清晰的「停止」音效，讓玩家立刻感知到輪到自己決定 |
| 自動戰鬥音效 | 槍聲、爆炸聲、環境聲，在背景持續進行 |
| 決策執行回饋 | 每個選項執行後有對應音效（盾撞門的金屬聲、炸彈爆炸聲等） |

## 十三、技術規格

| 項目 | 內容 |
|------|------|
| 引擎 | Godot 4 |
| 目標平台 | Android / iOS（優先），PC（次之） |
| 場景架構 | Isometric 3D（Camera3D 斜角固定） |
| 角色模型 | Meshy.ai 生成低多邊形 3D，.glb 格式匯入 |
| 資料格式 | JSON（角色設定、任務設定、決策點設定） |
| 存檔系統 | Godot 內建 ConfigFile，本機儲存 |
| UI 框架 | Godot Control 節點，CanvasLayer 疊加 |
| 目標幀率 | 60 fps（手機最低 30 fps 穩定） |

### MVP 技術優先順序

1. 小隊自動推進系統（路徑跟隨、隊列位置維持）
2. 決策點觸發與暫停系統
3. 選項 UI 彈出與選擇回饋
4. 4 個職業的自動行為 AI
5. 基本 Isometric 場景渲染

---

## 十四、MVP 範圍定義

**MVP 驗收目標：驗證「決策點中斷自動推進」的核心感受。**

### MVP 包含

- 1 張測試用地圖（3–4 個房間 + 岔路 + Boss 房）
- 4 個起始職業（基礎數值，Lv.1–3 決策選項）
- 3 種決策點類型（門、岔路、特殊事件 1 種）
- 自動戰鬥 AI（4 個職業各自的基本自動行為）
- 基本 HUD（血條、決策面板）
- 勝負判定 + 結算畫面

### MVP 不包含

- 抽卡系統（用預設固定陣容代替）
- 基地離線產金幣
- 主線故事 / 任務系統
- Lv.6 / Lv.9 高階決策選項
- 美術成品（使用佔位 3D 素材）
