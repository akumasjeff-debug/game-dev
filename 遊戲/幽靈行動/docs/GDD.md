## 二、世界觀與敘事

### 背景設定
玩家扮演一支**被政府正式切割的前特種部隊**，以「不存在」的身份繼續執行政府不想留下痕跡的任務。敵人涵蓋恐怖組織、腐敗警察、私人軍事公司、敵國特工，乃至政府自己派出的清除小隊。

隨劇情推進，玩家逐漸發現政府正在秘密進行的生化實驗——引出後期的非人類怪物關卡。

### 組織命名
玩家可自行為組織命名，或按骰子從改編真實特種部隊名稱的名稱池隨機抽取（約 15–20 個選項，如 Task Force Null、IXA Unit、Shadow Fox 等）。

### 敘事呈現
- 任務前：2–4 行文字情報說明（任務目標、背景）
- 任務後：結果畫面 + 解鎖下一段故事情報
- 任務內：無過場，節奏不中斷

## 三、核心操控

### 自適應雙模式（Adaptive Control）

系統根據右搖桿是否有輸入自動判斷模式，不需玩家手動切換：

| 狀態 | 行為 |
|------|------|
| 右搖桿有輸入 | 槍線跟隨右搖桿（專業模式） |
| 右搖桿無輸入 | 槍線跟隨移動方向（簡易模式） |
| 完全靜止 | 槍線自動瞄準視野內最近敵人 |

**PC 端**：WASD 移動、滑鼠位置控制槍線（永遠是專業模式）

### 安全模式
按 F 鍵 / 鎖頭按鈕切換：關閉自動射擊，進入匿蹤狀態。槍線接觸敵人不自動開槍，需手動扣板機。

## 四、視野與霧戰

- 每個角色的視野錐形方向 = 槍線方向，各成員獨立
- 全隊視野資訊共享（任一成員可見 = 全隊地圖更新）
- 成員間距過遠 → 失去視野共享

### 三層霧戰

| 狀態 | 顯示 |
|------|------|
| 未探索 | 全黑 |
| 探索過但不在視線內 | 灰霧（可看地形，看不到敵人） |
| 當前視線內 | 清晰 |

### 視野規格（依職業）

| 職業 | 錐角 | 距離 |
|------|------|------|
| 突擊手 | 120° | 中 |
| 狙擊手 | 60° | 極遠 |
| 盾兵 | 180° | 極短 |
| 支援/醫療兵 | 150° | 短 |
| 偵察手 | 90° | 遠 |
| 重裝 | 90° | 中 |

## 五、職業系統

共 8 種職業，進度解鎖（部分任務獎勵解鎖，部分花錢購買）。每種職業綁定專屬武器類型。

| 職業 | 專屬武器 | HP | 移速 | 特色 |
|------|---------|-----|------|------|
| 突擊手 | 步槍（AK / M4 系列） | 100 | 1.0x | 均衡主力 |
| 狙擊手 | 狙擊槍（AWP 系列） | 100 | 0.6x | 極遠射程，換彈三段極慢 |
| 盾兵 | 手槍 + 盾牌 | 150 | 0.5x | 正面 -85% 傷害 |
| 重裝 | 機槍（M249 / Negev） | 200 | 0.55x | 最強持續火力 |
| 醫療兵 | SMG | 120 | 0.85x | 急救更快、每局可復活一次倒地隊員 |
| 散彈兵 | 散彈槍 | 120 | 0.8x | 近距離極高傷害 |
| 偵察手 | 輕武器 | 90 | 1.0x | 視野最遠，匿蹤首選 |
| 爆破手 | 手槍 | 100 | 0.9x | 攜帶最多手榴彈，破壞掩體專家 |

### 複製職業規則
同一職業第 N 個成員的解鎖 EXP 需求：×1 / ×2 / ×4 / ×8

## 六、六大素質

每升一級獲得 3 點，自由分配。素質 8 級以上每點需消耗 2 點。

| 素質 | 影響效果 |
|------|---------|
| 體力 (Vitality) | HP 上限、倒地等待時間 |
| 敏捷 (Agility) | 移動速度、換彈速度 |
| 感知 (Perception) | 視野範圍、聽音距離 |
| 力量 (Strength) | 最大負重、重武器移速懲罰折減 |
| 技術 (Technique) | 槍線穩定度、道具使用效率 |
| 意志 (Willpower) | 自然回血速度、壓力懲罰折減 |

**每 5 級**解鎖一個技能槽，可裝備一個改變玩法的主動 / 觸發技能（非純數值加成）。

## 七、武器系統

### 武器來源（CS2 為設計基礎）

| 類型 | 代表武器 | 特性 |
|------|---------|------|
| 手槍 | Glock / Desert Eagle | 輕，無移速懲罰 |
| 衝鋒槍 | MP5 / P90 | 快，輕微移速懲罰 |
| 步槍 | AK-47 / M4A4 | 均衡 |
| 散彈槍 | Nova / XM1014 | 近距超高傷，射程極短 |
| 狙擊槍 | AWP / SSG 08 | 極遠，換彈極慢，移速最低 |
| 機槍 | M249 / Negev | 大彈匣，換彈極慢，移速最低 |

### 換彈機制
- 彈匣打空後**自動換彈**（不可中途換彈）
- 備彈無上限（彈藥管理不是瓶頸）
- 換彈期間完全無法射擊，是最大弱點視窗

### 武器升級（基地）
- 花金錢嘗試提升武器等級（機率制）
- 等級 1–5：高機率（60–80%）
- 等級 6–8：中機率（30–40%）
- 等級 9–10：低機率（10–15%），費用固定
- 失敗不扣等，上限 10 級

## 八、裝備槽

每個隊員：**1 把槍 + 1 個道具槽**

道具槽可裝備（全隊員共用上限 = 隊員人數）：
- 煙霧彈（落地 1s 後擴散，持續 10s，雙方視野中斷）
- 手榴彈（可預熱最多 3s，落地後 3s 爆炸）
- 醫療包（手動使用，自用或對隊友）

---

## 九、陣形系統

### 核心規則
- 出發前預設最多 **4 個陣形**
- 戰鬥中一鍵切換（PC：Q / E / Z / X）
- **全隊速度 = 主要位置角色的移速**
- 其他成員以自己的移速真實跟上（不瞬移），慢的成員自然落後

### 主要位置
- 鏡頭跟隨「主要位置」的角色
- 陣形中有明確標示哪個位置是主要位置
- 主要位置角色陣亡 → 自動切換最近存活隊員
- 玩家可在地圖上隨時手動切換誰是主要位置（用於選擇升級包等決策）

## 十、倒地與救援

| 階段 | 說明 |
|------|------|
| 倒地 | HP 歸零後進入倒地，不直接死亡 |
| 倒數計時 | 開始 10 秒倒數 |
| 救援 | 任何隊員靠近即可施救（醫療兵更快 + 救起後血量更多） |
| 救援中 | 施救者無法射擊，移出救援範圍即中斷 |
| 中斷 | 倒數繼續，需重新靠近 |
| 真死 | 倒數歸零 = 永久死亡（此局內） |

## 十一、匿蹤系統

### 警戒等級

| 等級 | 觸發 | 敵人行為 |
|------|------|---------|
| 安全 | 未被發現 | 固定巡邏路線 |
| 警戒 | 聽到聲音 / 短暫目擊 | 停止巡邏，搜索行為，視野擴大 |
| 戰鬥 | 確認發現玩家 | 主動攻擊，按警戒等級呼叫增援 |

- 警戒可隨時間消退（戰鬥→警戒需藏 5s，警戒→安全需藏 10s）
- 純視線迴避，無主動引誘機制
- 安全模式下槍線不自動射擊

### 屍體機制
敵人發現我方屍體 → 觸發全區搜索（警戒等級提升至戰鬥）

### 噪音等級

| 行為 | 噪音 | 影響範圍 |
|------|------|---------|
| 正常移動 | 無 | — |
| 衝刺 | 低 | 2 格 |
| 一般射擊 | 高 | 8 格 |
| 消音武器 | 低 | 3 格 |
| 手榴彈爆炸 | 極高 | 全地圖 |
| 煙霧彈 | 無 | 匿蹤友好 |

## 十二、地圖系統

### 地圖結構
- 一張大地圖 = 一種場景（醫院 / 辦公室 / 街道 / 公園 / 軍事設施等）
- **固定功能區域 + 隨機連接通道**
- 隨機特殊房間偶爾出現（如：囚犯可解救並加入隊伍）

### 地圖內容
| 要素 | 說明 |
|------|------|
| 大型物件 | 擋住移動路徑（醫療機器、大型家具） |
| 小型物件 | 可直接穿越（椅子、小箱子） |
| 可破壞掩體 | 木箱受彈後消失 |
| 補給箱 | 補血量 + 補充手榴彈/煙霧彈 |
| 升級包 | 提升主要位置角色一個等級（此局有效，結束後失效） |

## 十三、敵人系統

### 數值分層（三級）

| 等級 | 行為 |
|------|------|
| 雜兵 | 直線衝向玩家 |
| 老兵 | 會找掩體，觀察換彈時機再進攻 |
| 首領 | 固守位置，可呼叫增援，血量高 |

### 增援機制
進入戰鬥狀態後，根據警戒等級從地圖邊緣生成增援，迫使玩家不能龜縮。

### Boss
- 超高 HP（血量越低越兇）
- 操控小兵，同時處理多個目標

## 十四、任務系統

### 任務類型

| 類型 | 目標 | 失敗條件 |
|------|------|---------|
| 救援人質 | 找到 NPC 帶至出口（NPC 被動跟隨）| 人質死亡 |
| 暗殺目標 | 找到並消滅指定目標 | 目標逃脫 |
| 限時防守 | 在倒計時內守住據點 | 據點失守 |
| 殲滅 | 消滅地圖所有敵人 | 全員陣亡 |

### 任務板結構
- 基地有任務板，玩家自由選擇接哪個任務
- **主線**：固定難度，線性推進，完成解鎖下一段故事
- **支線**：各種難度（含隨機難度），重複接只給金錢
- 打不過主線 → 刷支線賺錢升裝

| 功能 | 說明 |
|------|------|

| 武器購買 | 用金錢購買新武器 |
| 武器強化 | 機率制升級武器等級（1–10 級） |
| 任務板 | 選擇主線 / 支線任務 |
| 組織命名 | 設定 / 更改組織名稱（骰子或自訂） |

## 十六、Roguelite 成長結構

### 永久成長（跨局保留）
- 金錢（輸了也能獲得）
- 已解鎖職業
- 已購買 / 升級的武器等級
- 已擴展的隊員格數（最多 4 格，花錢購買）
- 故事進度

### 局內成長（此局失效）
- 升級包（提升主要位置隊員等級，局結束歸零）
- 補給箱補充的彈藥與 HP

## 十七、後期內容：怪物關卡

### 觸發邏輯
主線推進至特定章節後，出現政府秘密實驗相關任務，引入非人類敵人。

### 兩種怪物類型

| 類型 | 偵測方式 | 特性 |
|------|---------|------|
| 生化實驗體 | 360° 聽覺，不依賴視線 | 只有兩個狀態：沉睡 / 狂暴 |
| 軍事機械 | 視覺 + 雷達，固定巡邏 | 需特定武器才能有效擊穿 |

生化體對噪音極度敏感，讓匿蹤機制轉為以「安靜移動」為核心。
