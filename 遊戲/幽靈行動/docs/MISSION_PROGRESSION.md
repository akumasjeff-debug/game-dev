# 幽靈行動 — 主線任務進度規劃

版本：1.0 | 日期：2026-06-21 | 負責人：關卡進度設計師

---

## 設計總覽

**玩家旅程核心節奏：**

```
任務 1（教：決策點）→ 任務 2（教：大招）→ 任務 3（教：6選4組隊）
      ↓ 刷支線，素材把核心陣容升到 Lv.3~4
任務 4（挑：隱藏事件，需特定角色達 Lv.6）→ 任務 5（考：組合能力）
```

**各任務建議首次通關等級（均為參與該任務的 4 名角色平均等級）：**

| 任務 | 平均等級 | 關鍵門檻 |
|------|---------|---------|
| 任務 1 | Lv.1~2 | 無，只要有盾兵即可 |
| 任務 2 | Lv.2~3 | 任一角色有大招可用即可 |
| 任務 3 | Lv.3 | 偵察手 Lv.3（解鎖「環境滲透」） |
| 任務 4 | Lv.4~5 | 至少一名角色 Lv.6（觸發隱藏事件） |
| 任務 5 | Lv.5~6 | 偵察手 Lv.3 + 突擊手 Lv.3 + 盾兵 Lv.3 |

## 主線任務設計

---

### 任務 1：「鑰匙的代價」

**背景故事**

一間廢棄工廠傳出訊號——情報顯示有一批失聯情報員的通訊器材藏在裡面。
幽靈小隊奉命潛入取回，任務看似簡單，但工廠內仍有留守的武裝警衛。
這是小隊第一次真正的行動，也是你第一次學著在槍聲中做決定。

**建議陣容**
- 盾兵（必帶，任何等級）+ 自選 3 名（Lv.1 即可）
- 建議：盾兵 + 突擊手 + 爆破手 + 醫療兵

**建議通關等級：** 全員 Lv.1~2（平均 Lv.1.5）

**地圖結構**

```
入口 → 走廊遭遇（第一個決策點）→ 岔路選擇 → 倉庫房間 → 終點撤離
```

**主要挑戰**
- 敵人數量少（4~6 名），屬性偏弱，主要考驗是「別一衝到底」
- 3 個決策點，每個都給玩家充足時間理解「暫停 = 你要做選擇」

**教學目標：「決策點暫停與選擇」**

設計細節：
1. 第一個決策點是「前方房間」：教玩家看三個選項的差異（直衝 / 靜悄悄 / 投擲炸彈）
2. 第二個是「岔路」：教玩家看情報標示（左路已知敵人多 / 右路敵人少）
3. 第三個是「補給箱」：教玩家三選一（補血 / 補炸彈 / 抽卡券），感受資源決策

教學提示文字（在決策面板上方小字標示）：
- 第 1 個決策點：「遊戲暫停了——輪到你做決定。選好按確認，任務繼續。」
- 之後不再重複提示

**通關獎勵**
- 金幣：800
- 素材：90 顆（指定素材，可升任意角色）
- 抽卡券：藍色票 x1
- 解鎖：故事片段「幽靈小隊第一份作戰報告」

**解鎖條件**
- 初始即可接取（遊戲開局任務）

**重玩動機**
- 補給箱三選一的「抽卡券」選項：首次通關建議選補血，重玩可試試抽卡券路線
- 任務本身短（約 3 分鐘），是最佳的素材刷新起點

**設計備注**
- 任務 1 的掉落素材 90 顆，配合起始的 180 顆（初始贈予），玩家在完成任務 1 後約有 270 顆素材，剛好可以把一名角色升到 Lv.3——這個時機點是刻意設計的，升完後立刻體驗 Lv.3 決策選項再進任務 2。

### 任務 2：「最後的窗口」

**背景故事**

郊區一棟被佔領的通訊大樓，敵方正在傳送一份加密檔案。
幽靈小隊有 10 分鐘的窗口可以突入、截取檔案、撤離——超時就算任務成功也會觸發大規模搜索，導致後續情報受損。
時間壓力迫使你必須用大招清場，而不是等自動推進慢慢磨。

**建議陣容**
- 任意 4 人，但須至少有一名大招 CD 短的角色（突擊手 CD 25 秒 / 偵察手 CD 35 秒最佳）
- 建議：突擊手 + 爆破手 + 狙擊手 + 盾兵

**建議通關等級：** 全員 Lv.2~3（平均 Lv.2.5）

**地圖結構**

```
底層入口 → 電梯井（狹道遭遇）→ 2F 走廊（大量敵人）→ 3F 伺服器室（Boss 護衛）→ 頂層撤離點
```

**主要挑戰**
- 敵人數量比任務 1 多出 60%（約 10~14 名）
- 2F 走廊有 3~4 名敵人同時出現——自動推進會打得很辛苦，但施放大招可以秒清
- 無時間限制（設計成情境壓力而非機制計時），玩家若慢打也能過，但會感受到「大招真的有差」

**教學目標：「大招系統——點卡施放與 CD 管理」**

設計細節：
1. 2F 走廊遭遇前，出現偵察手（NPC 角色）的對話提示：「那邊至少有 4 個，快上大招！」
2. 若玩家在走廊沒有施放任何大招，且戰鬥耗時超過 45 秒，畫面出現淡入提示：「點擊底部角色卡片可以施放大招」（僅顯示一次）
3. 伺服器室進入前有補給箱決策點，此時玩家的大招都在 CD 中——教玩家感受「CD 期間的壓力」

**通關獎勵**
- 金幣：1,200
- 素材：130 顆（指定素材）
- 抽卡券：藍色票 x2
- 解鎖：故事片段「截取的加密訊號——某個組織的名字首次出現」

**解鎖條件**
- 完成任務 1

**重玩動機**
- 伺服器室的「靜悄悄進入」選項：若帶了偵察手，可觸發小事件——偵察手潛入後發現有人在裡面等你，直接岔開別的路線。首次通關很難注意到這個分支。
- 刷素材效率比任務 1 好（素材量 +44%）

### 任務 3：「六個人，四個位置」

**背景故事**

情報確認了那個組織的一個前線基地。
此次任務規模較大，出發前指揮官給了你六名隊員的名單——但作戰車只能載四個人。
而且基地有電子干擾設備，偵察手的存在可能是突破這層屏障的關鍵。

**建議陣容**
- 偵察手（強烈推薦，本任務首次展示其核心價值）
- 偵察手 Lv.3 時「環境滲透」可在兩個關鍵房間提早揭露敵人配置
- 若不帶偵察手，仍可通關，但有兩個決策點會是「未知房間」且無預警

**建議通關等級：** 全員 Lv.3（偵察手 Lv.3 最重要）

**地圖結構**

```
基地外圍 → 檢查哨（選路：正面 / 側翼 / 偵察手先滲透）
         → 主樓底層（走廊 + 伏擊點）
         → 指揮室（隱藏情報決策點）
         → 基地後門（撤離）
```

**主要挑戰**
- 地圖最長（比前兩個任務多 2 個房間）
- 「伏擊點」是玩家第一次遇到伏擊事件：繼續推進 / 繞路 / 先投煙霧彈（三選一）
- 指揮室前有「岔路」決策，正確路線（側門）比直衝少遭一次戰鬥

**教學目標：「6 選 4 組隊策略 + 偵察手的預警功能」**

設計細節：
1. 任務開始前的「選人畫面」首次顯示完整 6 格職業欄位（之前只有 4 格），UI 提示：「選 4 名隊員出擊——每個職業都有自己的位置」
2. 若玩家選入偵察手，開局 5 秒後偵察手台詞：「我先探一探前面的狀況——注意看決策面板的預警圖示」
3. 偵察手的「預警」功能在本任務強制觸發一次（不靠機率），讓玩家確實看到「前方是伏擊」的預告文字
4. 帶偵察手且 Lv.3 時，「環境滲透」在檢查哨和伏擊點各觸發一次，玩家可以選「讓偵察手先滲透」

**通關獎勵**
- 金幣：1,800
- 素材：180 顆
- 抽卡券：藍色票 x2 + 金色票 x1
- 解鎖：故事片段「基地的檔案揭露了一個更大計畫的輪廓」

**解鎖條件**
- 完成任務 2

**重玩動機**
- 「不帶偵察手」挑戰：試試看只靠突擊手 + 爆破手 + 狙擊手 + 盾兵能否打穿
- 指揮室的「隱藏情報決策點」：若帶了偵察手 Lv.3，環境滲透後會發現指揮室有一台筆電——觸發「截取完整作戰計畫」，獲得額外金幣 500，且解鎖任務 5 的一條隱藏路線

### 任務 4：「層層設防」

**背景故事**

那份作戰計畫指向城郊一棟多層倉儲設施。
組織在這裡儲存了重要的協議文件，並布置了三層防線——包含一名他們的精英指揮官。
情報說這棟建築的某些角落很不尋常，但細節已被加密。幽靈小隊需要帶對人，才能看見那些角落。

**建議陣容**
- 至少一名角色達到 Lv.6（任意職業），才能觸發該職業的隱藏事件
- 若帶盾兵 Lv.6（「盾牆壓制」），第二層防線可以用側翼清掉 50% 敵人
- 若帶突擊手 Lv.6（「側翼突破」），隱藏武器庫事件可觸發
- 推薦組合：盾兵 Lv.5+ + 醫療兵 Lv.3+ + 突擊手 Lv.5+ + 狙擊手 Lv.3+

**建議通關等級：** 平均 Lv.4~5，至少一名角色 Lv.6

**地圖結構**

```
設施外圍 → 一層倉庫（大量低級敵人 × AOE 測試）
         → 二層走廊（精英敵人 × 盾牆壓制觸發點）
         → 三層控制室（精英指揮官 Boss 前廳）
         → 指揮官房間（Boss 戰）
         → 頂層天台（撤離）
```

**主要挑戰**
- 一層倉庫首次出現「多敵一波次」：6~8 名同時湧入，考驗 AOE 大招時機（爆破手大招是最佳解）
- 二層走廊出現第一個「精英敵人」：HP 是普通敵人 3 倍，自動推進效率很低——狙擊手大招或爆破手大招才能有效處理
- Boss 戰：指揮官有護衛（2 名），若帶狙擊手 Lv.9 可先消一個護衛（隱藏事件「高塔狙擊點」）

**教學目標：「隱藏事件——帶對人才看得見的第二層內容」**

設計細節：
1. 本任務刻意在三個位置預留「隱藏事件觸發點」，分別對應突擊手 Lv.6、盾兵 Lv.6、狙擊手 Lv.9
2. 沒帶對應角色 → 玩家只看到普通房間，不知道有隱藏內容
3. 帶了對應角色但未達等級 → 角色台詞：「這裡有點不對勁，但我現在看不清楚。」（暗示升等後再來）
4. 達到觸發等級 → 對話台詞：「等等，這邊——」然後觸發完整隱藏事件
5. 任務說明頁面下方：「情報顯示此設施有多處不明確區域——帶對人選才能探清。」（這是玩家唯一的提示）

**隱藏事件（本任務可觸發）**

| 隱藏事件 | 觸發條件 | 觸發位置 | 獎勵 |
|---------|---------|---------|------|
| 武器補給站 | 突擊手 Lv.6 | 一層倉庫角落 | 全隊本關攻擊 +5%，額外素材 30 顆 |
| 盾牆壓制 | 盾兵 Lv.6 | 二層走廊正面 | 清除 50% 二層敵人，跳過一場戰鬥 |
| 高塔狙擊點 | 狙擊手 Lv.9 | 三層控制室外 | Boss 戰前消滅一名護衛 |

**通關獎勵**
- 金幣：2,800
- 素材：250 顆
- 抽卡券：金色票 x2
- 解鎖：故事片段「協議文件揭示了一個人名——真正的幕後主謀」

**解鎖條件**
- 完成任務 3

**首次通關注意事項（給玩家的提示邏輯）**
- 若玩家全滅於一層倉庫（最常見的死點）：結算畫面顯示「大量敵人同時出現——試試爆破手的引爆炸彈大招？」
- 若玩家全滅於精英敵人：結算畫面顯示「精英敵人 HP 極高，集中點殺效果更好——狙擊手的精準鎖定大招可以立刻解決他」

**重玩動機**
- 高等角色的隱藏事件三種路線，一次通關最多觸發一種（觸發點互斥）
- 素材掉落最豐富的 5 關之一，是刷 Lv.5→6 里程碑費用的主要來源
- Boss 戰前若帶偵察手 Lv.6（電子干擾），可使指揮官護衛無法呼叫增援——這是第四條路線，本任務說明頁完全不提示

### 任務 5：「幽靈清算」

**背景故事**

幕後主謀的真實身分已確認——他正在一艘停靠在港口的貨輪上主持一場秘密會議。
這是唯一的機會。作戰計畫要求你同時做到三件事：突破外圍安保、在特定時間點發動大招壓制、並在主謀逃離前鎖定他。
帶錯人、用錯時機，你就只是衝進一艘空船。

**建議陣容（這關有正確答案，但不只一組）**
- 路線 A（速攻型）：偵察手 Lv.3+ + 突擊手 Lv.3+ + 爆破手 Lv.3+ + 狙擊手 Lv.3+
- 路線 B（穩健型）：盾兵 Lv.3+ + 醫療兵 Lv.3+ + 偵察手 Lv.3+ + 突擊手 Lv.3+
- 兩種路線到達 Boss 前的決策點後，需要做出不同的大招組合選擇

**建議通關等級：** 全員 Lv.5~6（平均 Lv.5.5）

**地圖結構**

```
港口登船點 → 貨艙層（複數敵人波次 + 岔路）
           → 甲板層（伏擊 + 計時炸彈事件）
           → 船艙走廊（精英護衛 × 2）
           → 主謀會議室（Boss 戰）
           → 逃生艙（撤離）
```

**主要挑戰**

1. 計時炸彈事件（本系列首次）
   - 甲板層發現計時裝置：爆破手拆除（耗時 10 秒，全隊靜止）/ 全隊撤離繞路
   - 若選「拆除」：拆除期間精英護衛從後方出現，需要大招應對
   - 若選「撤離」：繞路多一個房間，但 HP 損失少

2. 精英護衛 × 2（二選一分路）
   - 左路（2 名護衛一起）：帶爆破手大招可同時削弱
   - 右路（分開各 1 名）：帶狙擊手逐一點殺更有效率

3. Boss 戰：主謀（高HP + 2 護衛）
   - 護衛先清，否則 Boss 有護盾（護衛提供 Boss 防禦加成）
   - Boss 本體大招需要正確時機用盾兵護盾或偵察手 CC 應對

**教學目標：「組合考驗——正確組隊 + 正確大招時機 + 正確決策」**

設計細節：
1. 本任務不給教學提示——玩家用前 4 關學到的全部自行判斷
2. 計時炸彈強迫玩家在「資源消耗」和「時間風險」之間做取捨
3. Boss 護衛機制要求玩家先清小兵再打 Boss（正確決策順序）
4. Boss 戰中若沒有適時用大招應對 Boss 的出招，會有顯著的 HP 損失——體現「大招管理」的重要性

**通關獎勵**
- 金幣：4,500
- 素材：350 顆（高等素材，升 Lv.6~7 用）
- 抽卡券：金色票 x3
- 解鎖：故事片段「主謀落網，但他最後說的話讓你覺得這只是開始」
- 解鎖：後續任務板擴展（進入中期任務池）

**解鎖條件**
- 完成任務 4

**隱藏事件（本任務可觸發）**

| 隱藏事件 | 觸發條件 | 觸發位置 | 獎勵 |
|---------|---------|---------|------|
| 廢棄情報基地 | 偵察手 Lv.9 | 甲板層側門 | 揭露剩餘所有房間敵人配置 + 雙倍素材 |
| 傷患急救站 | 醫療兵 Lv.9 | 船艙走廊中段 | 全隊復活 + 任務後額外金幣 2,000 |

**重玩動機**
- 偵察手 Lv.9 隱藏路線（廢棄情報基地）讓任務素材翻倍——是後期刷素材的頂級選擇
- 計時炸彈的兩種選擇帶來截然不同的後半段體驗（HP 曲線完全不同）
- 前期不帶醫療兵的玩家，看到傷患急救站效果後，會開始考慮重組陣容

## 支線任務設計（素材刷新用）

支線任務可重複接取，無解鎖限制，隨任務板刷新出現。每種支線任務有自己固定的掉落素材池。

---

### 支線 A：「邊境哨站清除」

**背景簡述**
邊境的一個小型哨站，每週都有情報員在此被捕。幽靈小隊定期清除威脅，維持邊境的通道暢通。

**關卡特性**
- 敵人量少但密度高（小地圖 × 3 波）
- 無岔路，無特殊事件，純戰鬥
- 難度設計偏低，適合測試剛升級的角色

**建議等級：** Lv.1~3（入門刷素材任務）

**主要掉落**
- 通用素材：60~90 顆（每次略有浮動）
- 金幣：500~700

## 素材掉落與升等速度驗算

**前期（Lv.1 → Lv.3）需求：270 顆 / 角色**

| 來源 | 單次掉落 | 達到 270 顆需刷次數 |
|------|---------|------------------|
| 任務 1（主線） | 90 顆 | 加初始贈予 180 顆，1 次即達 |
| 支線 A | 60~90 顆 | 2~4 次 |

玩家在「完成任務 1 → 進任務 2」期間，有 1 名角色達到 Lv.3 的素材，符合 BALANCE_SHEET 的設計目標（2~3 場後解鎖第一個決策選項）。

**中期（Lv.3 → Lv.6）需求：1,320 顆 / 角色**

| 來源 | 單次掉落 | 累計達標次數 |
|------|---------|------------|
| 任務 2~3 各打一次 | 130 + 180 = 310 顆 | — |
| 支線 B（主力） | 120~160 顆 | 刷 7~9 次達 1,010 顆，搭配主線補足差額 |

玩家在任務 4 前應預期刷 6~10 次支線，這段時間讓他們熟練大招節奏。

**後期（Lv.6 → Lv.9）需求：5,400 顆 / 角色**

| 來源 | 單次掉落 | 累計達標次數 |
|------|---------|------------|
| 任務 4~5 各打一次 | 250 + 350 = 600 顆 | — |
| 支線 C（主力） | 150~220 顆 | 刷 21~26 次 |

後期刷支線 C 是長期目標，搭配主線 4、5 重玩可加速。預期玩家在此階段累積 30+ 小時遊玩時數。

## 待確認問題（供數值企劃對焦）

1. **初始贈予素材**：文件假設玩家初始有 180 顆素材，使任務 1 後可升到 Lv.3。若初始贈予不同，任務 1 的素材掉落需對應調整。
2. **支線 A 的機率性藍色票（25%）**：是否過高？初期若玩家快速累積藍色票，抽卡速度可能影響角色多樣性測試的時機。
3. **任務 4 的「至少一名 Lv.6」門檻**：若玩家只刷支線 B 而不打主線 2~3 的重玩，進入任務 4 時是否有足夠素材？建議確認「完成任務 1~3 + 支線 A×3 + 支線 B×6」的素材總量是否夠一名角色到 Lv.6。
