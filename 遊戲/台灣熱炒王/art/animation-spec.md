# 台灣熱炒王 — 角色動畫規格

**版本：** v1.0 | **日期：** 2026-06-20

---

## 全角色通用原則

- 角色尺寸：16x24 px（寬 x 高）
- 基礎格子對齊：16x16 px，腳底對齊格子下緣，頭部超出 8px
- 每幀時長預設：150ms（約 6.7fps，開羅感節奏）
- 精靈表排列：橫向展開每個狀態，每列一個方向或一種動作
- 透明色：左上角第一個像素作為透明色基準

## 角色動畫規格

### 老闆娘（管帳吸客）

| 動畫狀態 | 幀數 | 每幀時長 | 是否循環 |
|---------|------|---------|---------|
| Idle（待機，正面） | 2 幀 | 400ms / 600ms | 是 |
| Walk（行走，四方向各） | 4 幀 | 150ms | 是 |
| 點餐動作（拿筆記錄） | 4 幀 | 120ms | 是 |
| 吸客動作（揮手招攬） | 6 幀 | 130ms | 是 |
| 滿意（雙手叉腰點頭） | 3 幀 | 200ms | 否，播完回 Idle |
| 生氣（跺腳） | 4 幀 | 100ms | 否，播完回 Idle |

**Walk 正面（4 幀）：**
1. 右腳前伸 +2px，左手前擺 +1px
2. 雙腳並攏，身體上移 +1px（彈跳感）
3. 左腳前伸 +2px，右手前擺 +1px
4. 雙腳並攏，身體上移 +1px

**吸客動作（6 幀）：**
1. 右手自然下垂
2. 右手舉至腰部 +4px
3. 右手舉至肩部 +8px，手掌像素向外展開
4. 手在高點停留（時長加倍 260ms）
5. 手下降至腰部
6. 手回原位

### 廚師阿龍（炒菜）

| 動畫狀態 | 幀數 | 每幀時長 | 是否循環 |
|---------|------|---------|---------|
| Idle（待機，背面對灶） | 2 幀 | 500ms / 300ms | 是 |
| Walk（四方向各） | 4 幀 | 150ms | 是 |
| 炒菜（基本炒動作） | 6 幀 | 100ms | 是 |
| 炒菜（加火翻炒，大動作） | 4 幀 | 80ms | 是 |
| 起鍋（盛盤） | 5 幀 | 130ms | 否，完成後回 Idle |
| 滿意（舉鍋蓋點頭） | 3 幀 | 200ms | 否 |
| 生氣（用鍋鏟敲台） | 4 幀 | 100ms | 否 |

**炒菜基本（6 幀）：**
1. 雙手握鍋，鍋在正中（基準位）
2. 鍋往右傾 +2px，前臂向右 +1px
3. 鍋舉起 +3px，模擬拋起食材
4. 鍋傾左 -2px，食材下落
5. 回正中，鍋略低 -1px
6. 雙臂略收，重回幀 1

**炒菜大動作（4 幀，火大時觸發）：**
1. 鍋大幅左傾 -4px，身體微左傾
2. 鍋猛力右抛，上移 +5px，身體右傾
3. 鍋在最高點（+6px），身體前傾
4. 鍋落回，身體回正

## 動畫轉換規則

```
[觸發條件]                    →  [切換至動畫]
---------------------------------------------------
玩家未指派任務                →  Idle
獲得移動指令                  →  Walk（對應方向）
抵達工作位置                  →  對應工作動作
工作完成                      →  Idle 或 Walk（視下一指令）
客人耐心條 < 30%              →  Idle 急躁
客人收到餐點                  →  滿意表情爆出 → Idle 滿意
客人耐心條歸零                →  生氣爆出 → Walk 離開
客人吃完                      →  離開動作 → 消失
```

**轉換優先級：**
1. 生氣 / 離開（最高優先，可打斷任何動畫）
2. 工作動作
3. Walk
4. Idle（兜底）

## 建築動態規格

### 霓虹招牌閃爍（32x16 px，4 幀）

| 幀 | 說明 | 時長 |
|----|------|------|
| 1 | 全亮（#FF4444 / #FFFF00） | 300ms |
| 2 | 右半段熄滅 | 100ms |
| 3 | 全亮 | 300ms |
| 4 | 左半段熄滅 | 100ms |

### 炒菜台冒煙（8x16 px，6 幀，疊加層 60% 不透明）

煙霧顏色：#CCCCCC，每幀上升 Y -1px，加 ±1px sin 波擺動（振幅 ±1px，週期 6 幀）。

### 炭火明火（16x12 px，4 幀）

底部 #FF6600，邊緣 #FF2200，頂端黃白 #FFFF00。每幀左右搖擺，週期 320ms。

### 燈泡串搖擺（64x8 px，6 幀）

左右搖擺各 2px，週期約 1.05 秒。每 3 個完整循環後，隨機 1 顆燈泡閃亮（#FFFACD，持續 80ms）。

## 精靈表整合建議

**角色精靈表：** 每個角色獨立一張，建議尺寸 256x192 px（X 軸為幀序，Y 軸為動畫狀態）

**建築特效精靈表（集中一張 128x64 px）：**
- 列 1：霓虹招牌（4 幀 x 32px）
- 列 2：炒菜台冒煙（6 幀 x 8px）
- 列 3：明火動畫（4 幀 x 16px）
- 列 4：燈泡串搖擺（6 幀 x 64px）
