# 台灣熱炒王 — 專案規則與主題設定

**類型：** 開羅遊戲（Kairosoft）風格經營模擬

**風格定調：** 俯視角 2D 像素，台灣夜市霓虹感，親切熱鬧

## 世界觀與氛圍

玩家從一間小熱炒攤起家，逐步擴建成台灣最知名的熱炒王國。
從路邊攤 → 店面 → 連鎖 → 傳奇熱炒王。

### 核心台灣元素

**菜單（初期）**
- 三杯雞、炒蛤蜊、皮蛋豆腐、鹽酥雞
- 烤肉串（豬肉串、雞皮串、玉米筍）
- 炒空心菜、蒜炒高麗菜、薑絲大腸
- 台啤、米酒、保力達（飲料線）

**氛圍道具**
- 霓虹招牌、紅色塑膠椅、折疊桌
- 大型風扇、貼滿菜單的白板
- 瓦斯爐台、鐵板燒台、炭火烤爐

**員工角色**
- 老闆娘（管帳、吸引回頭客）
- 廚師阿龍（炒菜速度、料理品質）
- 外場小弟（送餐、招待效率）
- 打工仔（兼任切菜/洗碗，可升級）

**節日活動**
- 中秋烤肉節（8-9 月）
- 尾牙（12 月）
- 農曆新年（暫停營業但有紅包活動）
- 媽祖繞境（隨機事件，大量客人湧入）

## 開羅遊戲核心機制參考

- 俯視角建造，格子系統擺放設備與桌椅
- 客人 AI 自動進出、點餐、等待、付錢、離開
- 員工 AI 自動工作（廚師炒菜、外場送餐）
- 升級系統（設備升級、員工技能升級）
- 組合加成（特定菜色 + 特定設備 = 招牌菜 bonus）
- 年份推進（每年有目標，通關後解鎖新年份）
- 名聲系統（影響客流量與高消費客群比例）

## 設計原則

1. **台灣味優先** — 道具、人名、菜名、事件全部在地化
2. **熱鬧感** — 畫面永遠要有動態感（冒煙、人群、霓虹閃爍）
3. **輕鬆上手** — 前 2 年為教學，第 3 年開始有真實挑戰
4. **組合樂趣** — 多種菜色 + 設備 + 員工組合，鼓勵嘗試
5. **台灣節慶鉤子** — 節日活動讓玩家有理由定期回來
