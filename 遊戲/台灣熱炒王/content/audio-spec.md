## 整體音效方向

### 風格定位

台灣熱炒王的聽覺體驗以「親切熱鬧、不令人疲憊」為核心原則。玩家長時間遊玩時，音樂必須讓人感到舒適、帶有台灣夜市的人情味，而非刺激或緊張。

整體音樂語言：8-bit 像素音色為基底，融合台灣傳統民謠旋律元素（五聲音階、山地音樂節奏感），並以電子合成器與 Lo-fi 手法讓聲音現代化，不顯古老。

### 聽膩防護原則

- 所有 BGM 長度建議 90 秒以上（避免過短反覆）
- 每首 BGM 設計 2-3 個小變奏段落，讓循環有細微變化感
- UI 音效盡量輕薄，不蓋過遊戲事件音效
- 音量架構：BGM 預設 60%，音效 100%，兩者獨立控制

## 第一節：BGM 清單

---

### BGM-01 主選單

- 情緒：期待感、輕快、溫暖邀請
- 樂器組合：主旋律 8-bit 電子琴（帶輕微 chorus）/ 節奏 輕鼓點＋簡單 bassline / 背景和聲 鐘聲＋木琴點綴
- BPM 範圍：100-108
- 音樂風格：8-bit 台灣廟宇節慶風，旋律帶五聲音階，有類似「辦桌」開場感
- 關鍵感受：玩家看到標題畫面時應感到「這個遊戲看起來很好玩」、有一種夜市開攤前的期待感。旋律朗朗上口，讓玩家初次聽到就覺得熟悉。
- 循環點建議：第一個完整樂句結束後（約第 32 拍），旋律回到主題前的靜態空拍處做 seamless loop
- 音量建議：BGM 65%、SFX 100%（主選單 SFX 少，可略提 BGM）
- 參考曲風：《開羅遊戲》系列主選單風格 + 《Stardew Valley》輕快標題曲的溫暖感

---

### BGM-02 平日營業

- 情緒：忙碌但不焦慮、節奏穩定、有幹勁
- 樂器組合：主旋律 電子琴＋偶發吉他撥弦短句 / 節奏 中速鼓機（帶輕微台鼓感）＋低音走線 / 背景和聲 合成 pad 淡墊
- BPM 範圍：112-120
- 音樂風格：8-bit 與 Lo-fi 融合，主旋律帶台灣客家山歌的五聲音階轉折，但節奏偏現代電子
- 關鍵感受：玩家應感到「我在做事、但不緊張」，像是在一個熟悉的廚房裡穩定工作，知道今天客人不算太多，可以把每道菜做好。
- 循環點建議：16 小節後返回主旋律段落，loop 點在主旋律第一拍前的空拍（建議加 2 拍靜音緩衝）
- 音量建議：BGM 58%、SFX 100%
- 參考曲風：《開羅》美食王系列白天 BGM 風格 + 《Recettear》主題曲的穩定感

---

### BGM-03 週末旺季

- 情緒：興奮、快節奏、熱鬧喧囂中的爽快感
- 樂器組合：主旋律 帶失真的 8-bit 方波合成器 / 節奏 加速鼓機＋切分音 bass / 背景和聲 人聲取樣短句（「讚！」「上菜！」類型的短音節，不需真人錄音，可用合成音）＋背景鍋鏟聲節奏感取樣
- BPM 範圍：128-138
- 音樂風格：8-bit 電子舞曲（輕量），帶夜市廣播感，旋律比平日版更短促有力
- 關鍵感受：玩家應感到「客人來了！快點快點！」的衝刺感，但不是讓人焦慮的壓力，而是爽快的挑戰感。像是夜市人潮最旺的那個瞬間。
- 循環點建議：8 小節段落 loop，切點在大鼓重拍前（4/4 拍第 1 拍前）
- 音量建議：BGM 55%、SFX 100%（週末 SFX 頻繁，BGM 稍降）
- 參考曲風：《開羅》遊樂場系列旺季 BGM + 《Overcooked》緊張段落的活力感（但不要太緊張）

---

### BGM-04 節慶模式

- 情緒：歡慶、喜氣洋洋、帶有台灣本土節慶色彩
- 樂器組合：主旋律 8-bit 嗩吶音色（模擬） / 節奏 鑼鼓節奏型（八分音符強拍）＋合成打擊 / 背景和聲 錦瑟般的中高音弦樂 pad＋鞭炮短取樣點綴
- BPM 範圍：120-132
- 音樂風格：8-bit 台灣廟會電音風，旋律帶濃厚本土節慶感，類似迎神廟會但更電子化、更可愛
- 關鍵感受：玩家應立刻感覺到「現在是特殊日子！」的興奮感，如同真正在台灣過節一樣的喜氣，讓人期待節慶事件的到來。
- 循環點建議：以 8 小節鑼鼓過門作為 loop 點，鑼鼓最後一拍後回主題
- 音量建議：BGM 60%、SFX 100%
- 參考曲風：《桃太郎電鐵》節慶曲風 + 台灣電音廟會現場感

---

### BGM-05 深夜時段

- 情緒：安靜溫暖、懷舊、帶點孤獨但有人情味
- 樂器組合：主旋律 電吉他輕撥（清音，帶迴音）/ 節奏 極簡刷鈸＋低音悶鼓 / 背景和聲 鋼琴和弦淡入淡出＋遠方的蟬聲取樣（環境音，不是音樂）
- BPM 範圍：72-84
- 音樂風格：Lo-fi 夜晚爵士，帶台灣老歌吟唱感（如黃昏的故鄉情緒），旋律緩慢但不沉悶
- 關鍵感受：玩家在深夜繼續開店時，應感受到一種「還有幾個老主顧在，不急」的溫度，像是台灣老巷子裡最後一攤的燈光感。適合玩家放鬆操作、不被催促。
- 循環點建議：吉他最後一個音符 sustain 尾巴接 loop 回主題開頭，使用淡入/淡出避免 pop 聲
- 音量建議：BGM 70%、SFX 100%（深夜 SFX 少，BGM 稍提以填補空白感）
- 參考曲風：《Animal Crossing》深夜 BGM 的溫暖孤獨感 + Lo-fi 台灣老歌情緒

---

### BGM-06 建造模式

- 情緒：專注、輕鬆規劃感、期待改造結果
- 樂器組合：主旋律 木琴或鐵琴短句（輕盈） / 節奏 輕打擊（鈴鼓類）＋極簡 bass / 背景和聲 口哨短旋律＋環境靜默感（比其他場景安靜許多）
- BPM 範圍：95-105
- 音樂風格：輕量 8-bit 益智遊戲風，有點像在整理房間時哼的歌，讓人想動手擺弄
- 關鍵感受：玩家在建造模式中應感到「我在改造我的店，很好玩」的創造欲。音樂不能搶主角，因為這個場景玩家在思考布局，BGM 只是陪伴。
- 循環點建議：木琴旋律段落結束後的靜止拍（2 拍空拍），做 seamless loop
- 音量建議：BGM 50%、SFX 100%（建造 SFX 頻繁，BGM 要退到最後面）
- 參考曲風：《開羅》系列建設模式 BGM + 《Cities: Skylines》建設規劃安靜感

## 第四節：音效整合規範（給程式設計師）

---

### Godot 4 AudioStreamPlayer 使用建議

**節點類型選擇：**

| 用途 | 建議節點 |
|-----|---------|
| BGM（全域，不受位置影響） | `AudioStreamPlayer`（非 2D 版） |
| 一般 UI 音效 | `AudioStreamPlayer`（掛在 UI Layer 上） |
| 廚房動作音效（有空間感需求） | `AudioStreamPlayer2D`（掛在對應廚師/爐具節點上） |
| 環境循環音效（夜市雜音、炭火聲） | `AudioStreamPlayer`（設 `loop = true`） |

**Bus 架構建議（Godot AudioServer）：**

```
Master
├── BGM Bus（音量可獨立控制，建議預設 -6 dB）
├── SFX Bus（音量可獨立控制，建議預設 0 dB）
│   ├── Kitchen（廚房音效子分組）
│   ├── Service（外場音效子分組）
│   └── UI（UI 音效子分組）
└── Ambient Bus（環境音效，建議預設 -12 dB）
```

在 Project Settings > Audio 中設定以上 Bus，並讓各 AudioStreamPlayer 指向對應 Bus。

---

### BGM 淡入淡出設定

| 場景切換類型 | 建議做法 |
|-----------|---------|
| 場景主動切換（如：進入建造模式） | 淡出 1.0 秒 → 靜音 0.2 秒 → 淡入 1.0 秒 |
| 時段自動切換（如：深夜時段開始） | 淡出 2.0 秒 → 淡入 2.0 秒（交叉淡入淡出） |
| 節慶模式觸發（突然切換） | 淡出 0.5 秒 → 淡入 0.3 秒（快速，強調節慶驚喜感） |
| 遊戲暫停 | BGM 音量降至 30%（不完全靜音），不切換曲目 |

**實作方式（Godot 4 Tween）：**

```gdscript
# 淡出 BGM
var tween = create_tween()
tween.tween_property(bgm_player, "volume_db", -80, 1.0)
tween.tween_callback(bgm_player.stop)

# 淡入新 BGM
bgm_player.stream = new_bgm
bgm_player.volume_db = -80
bgm_player.play()
var tween2 = create_tween()
tween2.tween_property(bgm_player, "volume_db", -6, 1.0)
```

### 同時播放音效上限（手機效能）

| 類別 | 最大同時播放數 | 理由 |
|-----|-------------|-----|
| BGM | 1（同時最多 2，交叉淡入淡出期間） | 記憶體限制 |
| 廚房動作 SFX | 4 | 多個廚師同時工作 |
| 外場服務 SFX | 3 | 多桌同時服務 |
| UI SFX | 2 | 連點時避免爆音 |
| 環境音效 | 2 | 背景夜市音 + 一個特殊環境音 |
| **全域同時上限** | **10** | 保守值，避免中低階手機音頻卡頓 |

實作建議：使用 `AudioStreamPlayer` 的 `max_polyphony` 屬性控制；或自建 SFX Pool（AudioStreamPlayer 陣列），採用 round-robin 方式重用節點。

### 建議音效格式

**格式：OGG Vorbis**（.ogg）— Godot 4 的推薦格式，原生支援且解碼效率最佳。

| 類型 | 取樣率 | 位元率 | 說明 |
|-----|-------|-------|-----|
| BGM | 44100 Hz | 128-160 kbps | 音樂品質優先，檔案略大可接受 |
| 一般音效 | 44100 Hz | 96-128 kbps | 品質與效能的平衡點 |
| UI 音效（極短） | 22050 Hz | 64-96 kbps | 短音效降取樣率可大幅縮小檔案 |
| 環境循環音效 | 44100 Hz | 96 kbps | 需 seamless loop，取樣率不可降 |

**Loop 標記方式：** 在 Godot 4 中，OGG 檔案可透過 Import 面板設定 Loop（Edit > Reimport），不需要在音效本身嵌入 loop point，但建議音效本身剪接時已確保頭尾銜接自然。

### 靜音模式實作建議

玩家應能獨立控制：

1. **BGM 開/關**（Master BGM Bus 音量切換）
2. **音效 開/關**（Master SFX Bus + Ambient Bus 音量切換）

設定儲存於 `user://settings.cfg`，使用 Godot `ConfigFile` 存取。
建議在主選單提供明顯的音量控制旋鈕（BGM / SFX 兩條 slider），而非只有開關。
